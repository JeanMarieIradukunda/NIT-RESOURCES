📘 Converting a Python Program into a Standalone .exe

Using PyInstaller for Windows Applications

✅ Overview

PyInstaller is a tool that converts a Python script (.py) into a standalone executable file (.exe) for Windows.

  • The user does NOT need to install Python.
  • The program runs like normal software.

🔹 STEP 1 — Save Your Program

✔ What To Do:

  • Open Notepad or VS Code.
  • Paste your Python code.
  • Click Save As.
  • Name it: marks_converter.py
⚠ Important: The file must end with .py. Do NOT save it as .txt.

🔹 STEP 2 — Open Command Prompt

  • Go to the folder where the file is saved.
  • Hold Shift.
  • Right-click inside the folder.
  • Select "Open PowerShell window here".

Why? The command must run in the same folder as the file.

🔹 STEP 3 — Install PyInstaller

Run this command:

pip install pyinstaller

This installs the tool required to convert the program.

🔹 STEP 4 — Convert to .exe

Run:

pyinstaller --onefile marks_converter.py

Explanation:

  • pyinstaller → Runs the tool
  • --onefile → Creates one single executable
  • marks_converter.py → Your program

🔹 STEP 5 — Find the .exe File

  • After completion, two folders appear:
  • build
  • dist

Open the dist folder.

🎉 Inside you will find:

marks_converter.exe

This is your standalone program.

📌 What You Can Do Now

  • Copy the .exe file to another computer.
  • Run it directly.
  • No Python installation required.

✅ Optional (Cleaner Version)

Command:

pyinstaller --onefile --noconsole marks_converter.py

What it does:

Hides the black command window.

⚠ Your program uses input(), so do NOT use --noconsole for this program.

🔹 What Happens Internally?

  1. Bundles the Python interpreter.
  2. Includes your script.
  3. Adds required libraries.
  4. Creates a self-contained executable.

🎯 Final Result

  • Program runs like normal software.
  • No Python installation required.
  • Works on other Windows computers.
