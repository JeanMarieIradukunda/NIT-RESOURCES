Python Basic Concepts

Study Notes with Examples, Roles, and Real-Life Applications

1️⃣ Data Types

Definition: Data types define the kind of data a variable can hold and how Python handles it.

Type Example Description
int5Whole numbers
float5.6Decimal numbers
str"Hello"Text
boolTrueLogical values
list[1,2,3]Ordered collection
tuple(1,2,3)Immutable ordered items
dict{"name":"John"}Key-value pairs
set{1,2,3}Unordered unique items

Role: Organize memory, prevent errors, and guide operations on data.

Real-life Example: Different containers for rice, water, and money.

age = 25       # int
name = "Alice"   # str
price = 4.99     # float
is_paid = True   # bool

2️⃣ Variables

Definition: A variable is a named container for storing data in a program.

Rules to Declare:

  • Start with a letter or underscore
  • Can include letters, numbers, or underscores
  • Case-sensitive
  • No spaces or special characters
  • Use meaningful names

Modes of Variable Declaration:

  • Static: Value doesn’t change (pi = 3.14)
  • Dynamic: Value can change (age = 20; age = 21)
  • Global: Accessible everywhere (global x)
  • Local: Accessible inside function only

Real-life Example: Labels on containers (sugar, flour, rice).

student_name = "Iradukunda"
age = 20
print("Student:", student_name, "Age:", age)

3️⃣ Comments

Definition: Comments are non-executable lines that explain code.

Types:

  • Single-line: # This is a comment
  • Multi-line: '''This is a multi-line comment'''

Role: Improves readability, helps debugging, explains complex logic.

Real-life Example: Sticky notes or instructions on a machine.

# This program calculates area
length = 10
width = 5
area = length * width
"""The area is found by multiplying length and width."""
print("Area =", area)

4️⃣ Operators

Definition: Operators are symbols used to perform calculations, comparisons, or logical operations.

Type Description Example
ArithmeticMathematical calculations+ - * / %
AssignmentAssign values= += -=
ComparisonCompare values== != > < >= <=
LogicalCombine conditionsand, or, not
BitwiseWork on binary bits& | ^ << >>
MembershipCheck membershipin, not in
IdentityCompare memory locationis, is not

Role: Perform actions, comparisons, and logical decisions in code.

Real-life Example: Math symbols used to calculate total price in a supermarket.

a = 10
b = 3
print(a + b)          # 13
print(a > b)          # True
print(a > 5 and b < 5)  # True
