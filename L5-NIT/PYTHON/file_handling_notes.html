📘 ● File Handling in Python

File handling in Python is the process of working with files to store, read, write, update, or delete data permanently on a computer.
It allows programs to save information even after the program stops running.

🔹 Why File Handling is Important

  • Stores data permanently
  • Saves program output
  • Reads stored information
  • Manages large amounts of data
  • Used in real-world applications (databases, reports, logs, etc.)

🔹 Main Operations in File Handling

  1. Open a file
  2. Read from a file
  3. Write to a file
  4. Append data
  5. Delete a file
  6. Close a file

📂 ✔ File Handling Libraries

🔹 1. os Module

The os module is a built-in Python library that provides functions to interact with the operating system.

It allows Python programs to work with:

  • Files
  • Folders (directories)
  • Paths
  • System information

Example:

import os

print(os.getcwd())   # Current working directory

How to Use It

import os

Common os Functions

1️⃣ Get Current Working Directory

import os

print(os.getcwd())

2️⃣ Change Directory

os.chdir("folder_name")

3️⃣ List Files in a Folder

print(os.listdir())

4️⃣ Create a Folder

os.mkdir("new_folder")

5️⃣ Remove a File

os.remove("file.txt")

6️⃣ Remove an Empty Folder

os.rmdir("folder_name")

7️⃣ Check if File Exists

if os.path.exists("file.txt"):
    print("File exists")

🔹 2. pathlib Module

Pathlib is a built-in Python module that provides a modern and easy way to work with file and directory paths. It allows you to handle file paths using objects instead of simple strings. It was introduced in Python 3.4.

Why Use pathlib?

  • Easier to read and write
  • More organized code
  • Works across different operating systems (Windows, Linux, macOS)
  • Replaces many os.path functions

How to Use It

from pathlib import Path

Basic Example

from pathlib import Path

file = Path("example.txt")
print(file)

Common pathlib Operations

1️⃣ Check if File Exists

from pathlib import Path

file = Path("example.txt")

print(file.exists())

2️⃣ Create a File

from pathlib import Path

file = Path("newfile.txt")
file.touch()

3️⃣ Read a File

from pathlib import Path

file = Path("example.txt")
content = file.read_text()
print(content)

4️⃣ Write to a File

from pathlib import Path

file = Path("example.txt")
file.write_text("Hello World")

5️⃣ Get Current Directory

from pathlib import Path

print(Path.cwd())

🔹 3. shutil Module

shutil (short for shell utilities) is a built-in Python module used for high-level file and folder operations. It helps you copy, move, and delete files or directories easily.

Why Use shutil?

  • Copy files
  • Move files
  • Delete folders with content
  • Perform advanced file management tasks

How to Use It

import shutil

1️⃣ Copy a File

import shutil

shutil.copy("file1.txt", "file2.txt")

2️⃣ Move a File

import shutil

shutil.move("file1.txt", "folder/")

3️⃣ Delete a Folder with Content

import shutil

shutil.rmtree("folder_name")

File Modes in Python?

File modes tell Python how to open a file — whether to read, write, create, or append data. When using open(), you must specify a mode.

🔹 1️⃣ "r" — Read Mode

Meaning:
Opens a file for reading only.

Important:

  • File must already exist.
  • Cannot modify the file.
file = open("example.txt", "r")
print(file.read())
file.close()

🔹 2️⃣ "w" — Write Mode

Meaning:
Opens a file for writing.

Important:

  • Creates the file if it does not exist.
  • If the file exists, it overwrites (deletes old content).
file = open("example.txt", "w")
file.write("Hello")
file.close()

🔹 3️⃣ "x" — Create Mode

Meaning:
Creates a new file.

Important:

  • Gives an error if the file already exists.
  • Used when you want to ensure a new file.
file = open("newfile.txt", "x")
file.close()

🔹 4️⃣ "a" — Append Mode

Meaning:
Opens a file for adding data at the end.

Important:

  • Creates the file if it does not exist.
  • Does NOT delete existing content.
file = open("example.txt", "a")
file.write("\nNew Line")
file.close()

📘 ✔ Practice to Read File

🔹 1. Open File

Basic Syntax:

file = open("example.txt", "r")
  • "r" = read mode
  • File must exist

🔹 2. Read File

Read Entire File:

file = open("example.txt", "r")
content = file.read()
print(content)
file.close()

Read Line by Line:

file = open("example.txt", "r")
for line in file:
    print(line)
file.close()

🔹 3. Read File Permission

  • The file must exist.
  • You must use "r" mode.
  • You must have permission to access it.

If the file does not exist → Python gives an error.

📘 ✔ Perform Write/Create File

🔹 1. Create a New File

Using "x" mode:

file = open("newfile.txt", "x")
file.close()
  • Creates a new file.
  • Gives error if file already exists.

🔹 2. Write to Existing File

Using "w" mode:

file = open("example.txt", "w")
file.write("Hello World")
file.close()

⚠ Important:

  • "w" deletes old content.
  • It overwrites the file.

🔹 3. Append to Existing File

Using "a" mode:

file = open("example.txt", "a")
file.write("\nNew Line")
file.close()
  • Adds data without deleting old content.

🔹 Best Practice (Recommended)

Use with statement:

with open("example.txt", "r") as file:
    print(file.read())
  • Automatically closes the file.
  • Safer method.

📘 ✔ Perform Delete File

🔹 1. Remove File

import os

os.remove("example.txt")

🔹 Check Before Deleting:

import os

if os.path.exists("example.txt"):
    os.remove("example.txt")

🔹 2. Delete Folder

Delete Empty Folder:

import os

os.rmdir("myfolder")

Delete Folder With Files:

import shutil

shutil.rmtree("myfolder")

⚠ This permanently deletes everything inside the folder.

📊 Summary

✔ Reading Files

  • Use "r" mode
  • Use read() or loop

✔ Writing Files

  • "w" → Create/Overwrite
  • "a" → Append
  • "x" → Create new file

✔ Deleting

  • os.remove() → Delete file
  • os.rmdir() → Delete empty folder
  • shutil.rmtree() → Delete folder with content
