1️⃣ Python Generators

Exercise 1: Student ID Generator

A school wants to assign IDs to students sequentially as they enroll. Use a generator to create unique student IDs starting from 1001. Generate the first 10 IDs and print them.

def student_id_generator(start=1001):
    current = start
    while True:
        yield current
        current += 1

# Usage
gen = student_id_generator()
for _ in range(10):
    print(next(gen))
Hint: Use yield and a for loop to get IDs.

Exercise 2: Attendance Record Generator

Simulate marking attendance for 500 students, yielding one student at a time with the message: "Student {name} is present". Print the first 5 present students.

def attendance_generator(student_list):
    for student in student_list:
        yield f"Student {student} is present"

students = [f"Student{i}" for i in range(1, 501)]
gen = attendance_generator(students)

for _ in range(5):
    print(next(gen))
Hint: Use yield inside a loop iterating over student names.

2️⃣ Python Closures

Exercise 1: Grade Calculator Factory

Create a closure that generates a grade calculator for different exams. Then calculate the percentage score of a student who scored 42 in midterm (max 50) and 85 in final (max 100).

def grade_calculator(max_score):
    def calculate(score):
        return (score / max_score) * 100
    return calculate

midterm = grade_calculator(50)
final = grade_calculator(100)

print(midterm(42))  # 84%
print(final(85))    # 85%
Hint: Outer function takes max_score, inner function calculates percentage.

Exercise 2: Study Progress Tracker

Track cumulative study hours per day using a closure. Add new hours each day.

def study_tracker():
    total_hours = 0
    def add_hours(hours):
        nonlocal total_hours
        total_hours += hours
        return total_hours
    return add_hours

tracker = study_tracker()
print(f"Day 1: 2 hours → cumulative {tracker(2)}")
print(f"Day 2: 3 hours → cumulative {tracker(3)}")
Hint: Use nonlocal to update total hours in inner function.

3️⃣ Recursive Functions

Exercise 1: Total Assignment Scores

Calculate total score of assignments recursively, e.g., [8, 9, 7, 10].

def total_scores(scores):
    if not scores:
        return 0
    return scores[0] + total_scores(scores[1:])

print(total_scores([8, 9, 7, 10]))  # 34
Hint: Base case → empty list returns 0; recursive case → first element + rest.

Exercise 2: Factorial of Study Groups

Calculate how many ways to arrange 5 study groups using recursion.

def factorial(n):
    if n == 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))  # 120
Hint: Factorial(n) = n × Factorial(n-1), base case Factorial(1) = 1.
