Exercise 1: Student Marks Analyzer

Write a Python program that manages and analyzes student marks.

Requirement Done
Declare variables for student name, age, and marks using correct data types
Use comments to explain major parts of the code
Store marks in a list and student details in a dictionary
Create a user-defined function with a default pass mark (50)
Return the average score from the function
Use conditional statements to determine pass or fail
Use loop with continue for negative marks
Use break if a mark is greater than 100
Use at least one built-in function
Demonstrate use of pass for future feature
Call the function and display results
