📌 Project Question 1: Student Performance Analysis System
Scenario

Design a Python program that analyzes student performance data for a school.

Requirements
  • Use lambda functions to sort students by average score and filter pass/fail results.
  • Use generators to process large lists of student scores efficiently.
  • Implement closures to create subject-specific grading functions.
  • Apply decorators to log execution time and count function calls.
  • Use recursive functions to calculate cumulative statistics like total scores or averages.
Deliverables
  • Python source code
  • Sample input and output
  • Brief explanation of how each special function is used
📌 Project Question 2: Smart Task Automation & Monitoring System
Scenario

Create a Python-based system that automates and monitors tasks in a small organization.

Requirements
  • Use lambda functions to define task priorities and sorting logic.
  • Implement generators to yield scheduled tasks one at a time.
  • Use closures to generate task handlers with remembered configurations.
  • Apply decorators for authentication, logging, and error handling.
  • Use recursive functions to manage task dependencies.
Deliverables
  • Fully working Python program
  • Clear demonstration of each Python special function
  • Short documentation describing the system design
