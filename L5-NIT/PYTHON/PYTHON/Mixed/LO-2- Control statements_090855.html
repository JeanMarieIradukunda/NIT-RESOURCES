Applying Python Control Structures

1️⃣ Conditional Statements

Definition: Conditional statements allow a program to make decisions by executing certain blocks of code only when specific conditions are true.

Role: Control program flow, enable decision-making, automate choices.

Types:

  • if statement: Runs code if the condition is true. 
    age = 20
if age >= 18:
    print("You are eligible to vote.")
  • if-else statement: Runs one block if true, another if false. 
    number = 5
if number % 2 == 0:
    print("Even number")
else:
    print("Odd number")
  • if-elif-else statement: Handles multiple conditions. 
    marks = 75
if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
else:
    print("Grade C")
  • Nested if: One if inside another. 
    age = 25
if age > 18:
    if age < 30:
        print("Young Adult")
2️⃣ Looping Statements

Definition: Loops are used to execute a block of code repeatedly until a condition is met.

Role: Automate repetitive tasks, reduce code duplication, handle data efficiently.

Types:

  • for loop: Iterates over a sequence (list, string, range, etc.). 
    for i in range(1, 6):
    print("Number:", i)
  • while loop: Repeats as long as a condition is true. 
    count = 1
while count <= 5:
    print("Count:", count)
    count += 1
  • Nested loops: A loop inside another loop. 
    for i in range(1, 3):
    for j in range(1, 3):
        print(i, j)
3️⃣ Jump Statements

Definition: Jump statements control the flow inside loops by skipping, stopping, or doing nothing.

  • Break: Ends the loop immediately when a condition is met. 
    for num in range(1, 10):
    if num == 5:
        break
    print(num)  
# Stops when num == 5
  • Continue: Skips the current iteration and moves to the next. 
    for num in range(1, 6):
    if num == 3:
        continue
    print(num)  
# Skips printing 3
  • Pass: Acts as a placeholder; does nothing but keeps code valid. 
    for num in range(1, 6):
    if num == 3:
        pass  # to be implemented later
    print(num)
