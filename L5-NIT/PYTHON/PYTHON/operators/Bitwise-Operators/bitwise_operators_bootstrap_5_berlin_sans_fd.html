🧮 Bitwise Operators — Python
(Quick reference with examples)

Compact reference — AND, OR, XOR, NOT, Shifts

x = 5 (0101) y = 3 (0011)
& AND

Name: AND

Compares each bit of two numbers. If both bits are 1, result is 1, otherwise 0.

✅ Result: 1
Example
 x = 5   # (0101)
 y = 3   # (0011)
 print(x & y)
How it works
0101 (5)
& 0011 (3)
----
0001 (1)
| OR

Name: OR

Compares each bit of two numbers. If either bit is 1, result is 1.

✅ Result: 7
Example
 x = 5   # (0101)
 y = 3   # (0011)
 print(x | y)
How it works
0101
| 0011
----
0111 (7)
^ XOR

Name: XOR (Exclusive OR)

Result is 1 only if bits are different (one is 1, other 0).

✅ Result: 6
Example
 x = 5   # (0101)
 y = 3   # (0011)
 print(x ^ y)
How it works
0101
^ 0011
----
0110 (6)
~ NOT

Name: NOT (Complement)

Inverts all bits — changes 1 to 0 and 0 to 1. In Python `~x == -(x + 1)`.

✅ Result: -6
Example
 x = 5   # (0101)
 print(~x)
How it works
0101 (5)
----
1010 (-6) # two's complement
<< Left Shift

Name: Zero fill left shift

Shifts bits left, fills with zeros on the right. Each left shift multiplies by 2.

✅ Result: 20
Example
 x = 5   # (0101)
 print(x << 2)
How it works
0101 << 2 → 010100 (20)
Equivalent: 5 * (2^2) = 20
>> Right Shift

Name: Signed right shift

Shifts bits right. Each right shift divides by 2 (keeps sign for negatives).

✅ Result: 1
Example
 x = 5   # (0101)
 print(x >> 2)
How it works
0101 >> 2 → 0001 (1)
Equivalent: 5 // (2^2) = 1
