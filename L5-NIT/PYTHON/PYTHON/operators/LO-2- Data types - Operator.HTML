Applying Python Basic Concepts

1️⃣ Data Types

a. Definition: Data types define the kind of values a variable can hold.

b. Role: Ensure correct operations, help manage memory, guide data usage.

c. Real-world Examples:

  • Integer: age = 20 → counting people in a class.
  • Float: temperature = 36.6 → measuring body temperature.
  • String: name = "Alice" → storing student’s name.
  • Boolean: is_logged_in = True → login status.
  • List: scores = [75, 80, 90] → exam results.
2️⃣ Variables

A. Definition: A variable is a named storage for holding data.

B. Roles: Store/reuse values, improve readability, enable data manipulation.

C. Rules:

  • Start with letter or underscore (_name, student1).
  • Cannot start with number (1student ❌).
  • Case-sensitive (Name ≠ name).
  • No spaces allowed (student name ❌).
  • Avoid reserved keywords (if, class).

Example:

student_name = "John"
student_age = 21
print(student_name, "is", student_age, "years old.")
3️⃣ Comments

Definition: Non-executable notes in code to explain logic.

Role: Improve readability, debugging, documentation.

Syntax:

# Single-line comment

"""
Multi-line comment
for longer explanations
"""

Example:

# Calculate area of rectangle
length = 10
width = 5
area = length * width
print(area)
4️⃣ Operators

Definition: Operators are symbols that perform specific operations on values or variables.

Types with Descriptions & Examples:

  • Arithmetic Operators: Used to perform basic mathematical calculations like addition, subtraction, multiplication, and division.
    + - * / % ** // 
    price = 50
quantity = 3
total = price * quantity  # Multiplication
print(total)  # 150
  • Comparison Operators: Used to compare two values and return a boolean result (True/False).
    == != > < >= <= 
    age = 18
print(age >= 18)  # True → checks if age is 18 or more
  • Logical Operators: Used to combine conditional statements and return True or False based on logic.
    and, or, not 
    student = True
fees_paid = True
print(student and fees_paid)  # True → both conditions must be true
  • Assignment Operators: Used to assign values to variables or update them with operations.
    = += -= *= /= 
    score = 10
score += 5  # same as score = score + 5
print(score)  # 15
  • Membership Operators: Used to test whether a value exists within a sequence (list, string, tuple).
    in, not in 
    names = ["Alice", "Bob", "John"]
print("Bob" in names)  # True → Bob is in the list
