Python Assignment Operators

1. = (Simple Assignment)

Assigns a value to a variable.

x = 5
print(x)  # Output: 5

Explanation: The value 5 is stored in the variable x.

2. += (Add AND Assign)

Adds the right-hand value to the variable and assigns the result.

x = 5
x += 3  # equivalent to x = x + 3
print(x)  # Output: 8

3. -= (Subtract AND Assign)

Subtracts the right-hand value from the variable and assigns the result.

x = 5
x -= 2  # equivalent to x = x - 2
print(x)  # Output: 3

4. *= (Multiply AND Assign)

Multiplies the variable by the right-hand value and assigns the result.

x = 5
x *= 3  # equivalent to x = x * 3
print(x)  # Output: 15

5. /= (Divide AND Assign)

Divides the variable by the right-hand value (floating-point division) and assigns the result.

x = 10
x /= 4  # equivalent to x = x / 4
print(x)  # Output: 2.5

6. %= (Modulus AND Assign)

Takes modulus of the variable with the right-hand value and assigns the remainder.

x = 10
x %= 3  # equivalent to x = x % 3
print(x)  # Output: 1

7. //= (Floor Division AND Assign)

Divides and rounds down to the nearest integer, then assigns the result.

x = 10
x //= 3  # equivalent to x = x // 3
print(x)  # Output: 3

8. **= (Exponent AND Assign)

Raises the variable to the power of the right-hand value and assigns the result.

x = 2
x **= 3  # equivalent to x = x ** 3
print(x)  # Output: 8

9. &= (Bitwise AND AND Assign)

Performs bitwise AND between the variable and the right-hand value.

x = 5      # binary: 0101
x &= 3     # binary: 0011
print(x)   # Output: 1 (0101 & 0011 = 0001)

10. |= (Bitwise OR AND Assign)

Performs bitwise OR and assigns the result.

x = 5      # binary: 0101
x |= 3     # binary: 0011
print(x)   # Output: 7 (0101 | 0011 = 0111)

11. ^= (Bitwise XOR AND Assign)

Performs bitwise XOR and assigns the result.

x = 5      # binary: 0101
x ^= 3     # binary: 0011
print(x)   # Output: 6 (0101 ^ 0011 = 0110)

12. >>= (Right Shift AND Assign)

Shifts the bits of the variable to the right and assigns the result.

x = 8      # binary: 1000
x >>= 2
print(x)   # Output: 2 (1000 >> 2 = 0010)

13. <<= (Left Shift AND Assign)

Shifts the bits of the variable to the left and assigns the result.

x = 2      # binary: 0010
x <<= 3
print(x)   # Output: 16 (0010 << 3 = 10000)

✅ Summary:

= → assign

+=, -=, *=, /=, %=, //=, **= → arithmetic + assign

&=, |=, ^=, >>=, <<= → bitwise + assign
