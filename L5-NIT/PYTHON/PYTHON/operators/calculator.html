Simple Calculator Code

Understand how a simple calculator works step by step 💡

print("Simple Calculator")
                        print("Enter number → operator → next number → ... → = to finish")

                        # Step 1: get the first number
                        first = float(input("Enter first number: "))
                        result = first

                        while True:
                            op = input("Enter operator (+, -, *, / or = to stop): ")

                            if op == "=":
                                print("Final Answer =", result)
                                break

                            next_num = float(input("Enter next number: "))

                            if op == "+":
                                result = result + next_num
                            elif op == "-":
                                result = result - next_num
                            elif op == "*":
                                result = result * next_num
                            elif op == "/":
                                result = result / next_num
                            else:
                                print("Invalid operator!")

                            print("Current Result =", result)

🧩 Now Let’s Explain It Clearly 👇

🟦 Line 1–2
print("Simple Calculator")
print("Enter number → operator → next number → ... → = to finish")

💬 These two lines just display messages to tell the user what the program does and how to use it.

🟦 Step 1: Get the first number
first = float(input("Enter first number: "))

👉 input() asks the user to type something.
👉 float() converts it to a number (with decimals).
👉 The result is stored in first.

🟦 Step 2: Store that number in result
result = first

This variable result will keep updating as we perform more operations.

🟦 Step 3: Start a loop to keep calculating
while True:

💡 Means “repeat again and again until we tell it to stop.”

🟦 Step 4: Ask for an operator
op = input("Enter operator (+, -, *, / or = to stop): ")

The user types one of these: +, -, *, / or =

🟦 Step 5: If user types "="
if op == "=":
    print("Final Answer =", result)
    break

If the user enters "=", the program shows the final result and stops the loop.

🟦 Step 6: Get the next number
next_num = float(input("Enter next number: "))

The user types another number to continue the calculation.

🟦 Step 7: Perform the correct operation
if op == "+": result = result + next_num
elif op == "-": result = result - next_num
elif op == "*": result = result * next_num
elif op == "/": result = result / next_num
else: print("Invalid operator!")

🧮 The correct math operation is done depending on the operator entered.

🟦 Step 8: Show the result after each operation
print("Current Result =", result)

Displays the updated total after every operation.
