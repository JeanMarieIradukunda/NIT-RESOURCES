Jump Statements in Python

break · continue · pass — what they do and practical examples
Python Concepts

Jump statements in Python are special control-flow statements that alter execution in loops or conditional blocks. They are mainly used to:

  • Exit a loop prematurely
  • Skip specific iterations
  • Write placeholder code without affecting execution
🔹 break Statement
Terminate a loop immediately when a condition is met.
Syntax
Example 
for i in range(1, 6):
    if i == 3:
        break
    print(i)
Output
1
2

When i == 3, the loop stops immediately — values 3, 4, 5 are not processed.

🔹 continue Statement
Skip the current iteration and continue with the next one.
Syntax
Example 
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
Output
1
2
4
5

When i == 3, that iteration is skipped; the loop continues with i = 4.

🔹 pass Statement
A no-op placeholder — useful when syntax requires a statement but logic is pending.
Syntax
Example 
for i in range(1, 6):
    if i == 3:
        pass  # placeholder for future code
    print(i)
Output
1
2
3
4
5

pass does nothing — control flows normally.

Practical Examples — break, continue, pass
Real-life usage
break — Stop Searching

Stop once the item is found.

students = ["Alice", "John", "Peter", "Maria", "David"]

search_name = "Maria"

for name in students:
    if name == search_name:
        print(f"{search_name} found in the list!")
        break
    print(f"Checking {name}...")
Output:
Checking Alice...
Checking John...
Checking Peter...
Maria found in the list!
continue — Skip Certain Values

Skip items that don't apply.

students = ["Alice", "John", "Peter", "Maria", "David"]

for name in students:
    if name == "John":
        continue  # skip John
    print(name)
Output:
Alice
Peter
Maria
David
pass — Placeholder for Future Logic

Leave a placeholder while continuing normal flow.

students = ["Alice", "John", "Peter"]

for name in students:
    if name == "John":
        pass  # will handle absence logic later
    else:
        print(f"{name} is present.")
Output:
Alice is present.
Peter is present.
