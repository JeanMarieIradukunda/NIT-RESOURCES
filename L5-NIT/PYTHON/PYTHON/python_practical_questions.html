🧠 Python Practical Questions

Section A — Applying Python Basic Concepts
Focus: Data Types, Variables, Comments, Operators

Exercises overview

1. Data Types (int, float, string, list, boolean)

(Exclude: dictionary, tuple, complex)

  1. Store and show types

    Store your name, age, and height in variables. Print them with their types using type().

  2. Subjects list

    Create a list of 5 subjects. Print the first and last subject. Replace the third subject and print the updated list.

  3. Even or odd

    Take a number from the user and check whether it's even or odd.

  4. Boolean message

    Create a variable is_student (boolean). Print: "Are you a student? True"

  5. Product prices

    Create two floats for product prices, add them and print the total cost.

2. Variables

  1. Swap values

    Swap two variables without using a third variable.

  2. Area of rectangle

    Create length and width. Calculate and print the area.

  3. Full name

    Store first and last name. Concatenate to display full name.

  4. Pay calculation

    Variables: hours_worked, rate_per_hour, total_pay. Compute and print total_pay.

  5. Assignment order

    Explain what happens if you assign a value to a variable before declaring it, then show a short example.

3. Comments

  1. Area of a circle

    Write a program to calculate the area of a circle. Include single-line and multi-line comments explaining the steps.

  2. Comment each line 
    name = "Alice"
age = 20
print("Hello", name, "you are", age, "years old")

    Use comments to describe what each line does.

  3. Three types of comments

    Create a script demonstrating: single-line, multi-line, and inline comments.

4. Operators (Arithmetic, Assignment, Comparison, Logical, Membership)

  1. Arithmetic operations

    Take two numbers as input and print their sum, difference, product, and quotient.

  2. Assignment operators

    Demonstrate +=, -=, *=, /= using x = 10.

  3. Compare numbers

    Write a program that prints whether two numbers are equal or not.

  4. Logical operators

    Using and, or, not, check if a student passes: pass if marks ≥ 50 and attendance ≥ 75%.

  5. Membership operator

    Check if a word exists in a list using in.

Bonus: Integrative Challenge

Combine all concepts: ask the user for their name, age, and 3 favourite subjects. Store them with correct types, calculate age in 5 years, and print a message:

"Hello <name>, in 5 years you will be <age+5> years old, and your first subject is <subject1>."
