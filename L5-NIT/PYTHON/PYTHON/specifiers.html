Python Format Specifiers

Definition: A specifier tells Python how to display or format a value in output. It controls type, precision, and alignment.

Common Format Specifiers

Specifier Meaning Example Output
%d Integer "%d" % 5 5
%f Floating-point number "%.2f" % 5.678 5.68
%s String "%s" % "Hello" Hello
:.2f 2 decimal places (f-strings) f"{3.14159:.2f}" 3.14
:>5 Right align, width 5 f"{7:>5}" " 7"
:<5 Left align, width 5 f"{7:<5}" "7 "

School Example: Student Marks

Python Code:

# Student information
name = "John"
math = 78.567
english = 85.4
science = 92.125

# Using f-strings with specifiers
print(f"Student Name: {name}")
print(f"Math Score: {math:.1f}")       # 1 decimal place
print(f"English Score: {english:.0f}") # Rounded to nearest integer
print(f"Science Score: {science:.2f}") # 2 decimal places

# Table format
print("\n{:<10} {:>10} {:>10} {:>10}".format("Name", "Math", "English", "Science"))
print("{:<10} {:>10.1f} {:>10.0f} {:>10.2f}".format(name, math, english, science))

Output:

Student Name: John
Math Score: 78.6
English Score: 85
Science Score: 92.13

Name           Math    English    Science
John           78.6         85      92.13
