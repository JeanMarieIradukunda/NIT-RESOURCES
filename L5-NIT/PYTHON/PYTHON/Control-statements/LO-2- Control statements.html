Applying Python Control Structures

1️⃣ Conditional Statements

Definition: Conditional statements allow a program to make decisions by executing certain blocks of code only when specific conditions are true.

Role: Control program flow, enable decision-making, and automate choices.

Types (with real-life student examples):

  • ✅ if statement: Runs code if the condition is true.
    💡 Example: Checking if a student is old enough to register for college. 
    age = 20

if age >= 18:
    print("✅ You are eligible to register for college.")

    Output:
    ✅ You are eligible to register for college.

  • 🔁 if-else statement: Runs one block if true, another if false.
    💡 Example: Checking if a student passed or failed an exam. 
    marks = 48

if marks >= 50:
    print("🎉 You passed the exam!")
else:
    print("😢 You failed. Please try again.")

    Output:
    😢 You failed. Please try again.

  • 📊 if-elif-else statement: Handles multiple conditions.
    💡 Example: Determining a student’s grade. 
    marks = 75

if marks >= 90:
    print("🏆 Grade A - Excellent!")
elif marks >= 75:
    print("👏 Grade B - Good job!")
elif marks >= 60:
    print("🙂 Grade C - Keep improving!")
else:
    print("❌ Grade F - Needs improvement.")

    Output:
    👏 Grade B - Good job!

  • 🪜 Nested if: One if inside another.
    💡 Example: Checking a student’s eligibility for a scholarship. 
    age = 19
marks = 92

if age >= 18:
    if marks >= 90:
        print("🎓 You are eligible for the scholarship!")
    else:
        print("👍 You meet the age requirement, but need higher marks.")
else:
    print("❌ You are too young to apply for the scholarship.")

    Output:
    🎓 You are eligible for the scholarship!

2️⃣ Looping Statements

🧠 Definition: Loops are used to repeat a block of code multiple times until a certain condition is met. They help us avoid writing the same code again and again.

🎯 Role: Automate repetitive tasks, reduce code duplication, and handle large data efficiently.

🌀 Types of Loops with Real-Life Student Examples

  • 1️⃣ for loop: Iterates over a sequence such as a list, string, or range.

    📘 Example: Displaying a Student’s Exam Numbers

    for i in range(1, 6):
    print("Exam paper number:", i)

    🧩 Explanation: A student has 5 exam papers. Instead of printing 1–5 manually, the for loop prints all numbers automatically.

    👩‍🎓 Real-life scenario: You are checking your assignments from Week 1 to Week 5 — the loop helps you repeat the task for each week.

  • 2️⃣ while loop: Repeats as long as a condition is true.

    📘 Example: Counting Study Hours

    hours = 1
while hours <= 5:
    print("Hour", hours, "- Keep studying! 📚")
    hours += 1

    🧩 Explanation: The loop continues until you’ve studied for 5 hours.

    👨‍🎓 Real-life scenario: You keep studying until your study hours reach your goal (e.g., 5 hours).

  • 3️⃣ Nested loops: A loop inside another loop.

    📘 Example: Generating a Timetable (Days × Subjects)

    days = ["Monday", "Tuesday"]
subjects = ["Math", "Science"]

for day in days:
    for subject in subjects:
        print(day, "→", subject)

    🧩 Output:

    Monday → Math
Monday → Science
Tuesday → Math
Tuesday → Science

    👩‍🏫 Real-life scenario: For each school day, you attend multiple subjects — a nested loop helps organize this timetable.

🏁 Summary Table

Loop Type Purpose Example Use Case

Nested loops Repeat within another repeat Creating timetable, seat arrangement
3️⃣ Jump Statements

Definition: Jump statements control how loops behave — they can stop a loop early, skip certain steps, or act as placeholders for future code.

  • Break: Stops the loop completely when a specific condition is met.
    🎓 Example: Stop checking students once an absent student is found. 
    students = ["Alice", "Brian", "Cathy", "Absent", "David"]

for student in students:
    if student == "Absent":
        print("⚠️ Attendance stopped! Found an absent student.")
        break
    print(f"✅ {student} is present.")
# Loop ends immediately after finding "Absent"
  • Continue: Skips one loop cycle and continues with the next iteration.
    📚 Example: Skip subjects that have already been graded. 
    subjects = ["Math", "Science", "English", "Done", "ICT"]

for subject in subjects:
    if subject == "Done":
        continue  # Skip this subject
    print(f"📖 Grading {subject}...")
# "Done" subject is skipped
  • Pass: Does nothing — used as a placeholder where code will be added later.
    🧠 Example: A student project not yet implemented but needs to stay valid. 
    projects = ["AI", "Web Design", "Data Science"]

for project in projects:
    if project == "Web Design":
        pass  # will add project details later
    print(f"📘 Project: {project}")
# Loop runs normally; 'pass' just keeps code structure valid
