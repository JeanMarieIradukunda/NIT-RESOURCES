📘 Python Example: Menu Selection Using match case

Scenario: A student selects an option from a school menu.

  • Step 1: Define functions for each menu option:
    • def view_home():
    print("🏠 Welcome to the Home Page!")

def show_profile():
    print("👤 User Profile:")
    print("Name: John Doe")
    print("Email: john@example.com")

def settings():
    print("⚙️ Settings Menu: Change your preferences here.")

def exit_program():
    print("👋 Exiting the program... Goodbye!")
  • Step 2: Display the menu:
    • print("📱 MAIN MENU")
print("1. Home")
print("2. Show Profile")
print("3. Settings")
print("4. Exit")
  • Step 3: Get user input and use match-case:
    • choice = int(input("Enter your choice (1-4): "))

match choice:
    case 1:
        view_home()
    case 2:
        show_profile()  # ← case 2: show_profile()
    case 3:
        settings()
    case 4:
        exit_program()
    case _:
        print("❌ Invalid option! Please select between 1 and 4.")
  • Step 4: Example Output:
    • If user enters 2:
👤 User Profile:
Name: John Doe
Email: john@example.com
