🐍 Python Loops Explained

🔁 What is a Loop?

In Python programming, a loop is a control structure used to repeat a block of code multiple times — either a fixed number of times or until a certain condition is met.

Python mainly provides two types of loops:

  • for Loop
  • while Loop

🔁 1. for Loop

Used when you know how many times you want to repeat a block of code.

Syntax:
for variable in sequence:
    # code to execute
Example:
for i in range(5):
    print("Hello", i)
👉 Output:
Hello 0
Hello 1
Hello 2
Hello 3
Hello 4

Here, the loop runs 5 times, because range(5) gives numbers from 0 to 4.

🔁 2. while Loop

Used when you don’t know in advance how many times to repeat — it keeps running as long as a condition is True.

Syntax:
while condition:
    # code to execute
Example:
count = 1
while count <= 5:
    print("Count:", count)
    count += 1
👉 Output:
Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

⚙️ Loop Control Statements

Python provides special keywords to control the flow of loops:

Keyword Description
breakStops the loop completely
continueSkips the current iteration and goes to the next one
passDoes nothing — acts as a placeholder
Example with break and continue:
for i in range(10):
    if i == 5:
        break   # Stop the loop when i is 5
    if i == 2:
        continue  # Skip printing 2
    print(i)
👉 Output:
0
1
3
4

✅ Summary

Loop Type When to Use Example
for loop You know how many times to repeat Iterating over a list or range
while loop You want to repeat until a condition changes Waiting for user input

🎓 Example 1: Looping through a List of Students

Suppose you have a list of students’ names:

students = ["Alice", "Brian", "Clara", "David"]

for student in students:
    print("Student Name:", student)
👉 Output:
Student Name: Alice
Student Name: Brian
Student Name: Clara
Student Name: David

🧠 Explanation: The loop goes through each element in the students list and prints it one by one.

👨‍🏫 Example 3: Asking for Attendance (using while loop)

A while loop is useful when you want to keep asking for input until a condition is met.

present_students = []

while True:
    name = input("Enter student name (or 'done' to finish): ")

    if name.lower() == 'done':
        break  # Stop the loop
    present_students.append(name)

print("Attendance List:", present_students)
👉 Example Output:
Enter student name (or 'done' to finish): Alice
Enter student name (or 'done' to finish): Brian
Enter student name (or 'done' to finish): done
Attendance List: ['Alice', 'Brian']

🧠 Explanation: This loop continues to ask for names until the user types 'done'.
