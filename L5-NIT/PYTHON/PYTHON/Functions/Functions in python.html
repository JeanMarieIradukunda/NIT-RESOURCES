🧠 Applying Functions in Python

Functions in Python help you organize code, avoid repetition, and make programs easier to read and maintain.

✔ Definition of Function

A function is a reusable block of code that performs a specific task.

def greet():
    print("Hello! Welcome to Python.")

Call it using:

greet()

✔ Types of Functions

1️⃣ Built-in Functions

Already made by Python. Examples: print(), len(), sum().

numbers = [2, 4, 6, 8]
print(len(numbers))   # 4
print(sum(numbers))   # 20
Analogy: Like the buttons on a calculator — already ready to use.
2️⃣ User-Defined Functions

Functions you create to perform specific tasks.

def add_numbers(a, b):
    total = a + b
    return total

print(add_numbers(5, 3))   # 8
Analogy: Like writing your own recipe for a dish.

✔ Creating a Function

def say_hello(name):
    print("Hello", name)

✔ Function Arguments

Data passed into a function to use in its task.

def greet_user(username):
    print("Good morning,", username)

greet_user("Alice")
greet_user("Bob")
Analogy: Dial different numbers on your phone to call different people, using the same phone.

✔ Default Parameter Value

def greet(name="Friend"):
    print("Hello,", name)

greet("John")
greet()   # uses default
Analogy: Like a vending machine giving water by default if you don’t select a drink.

✔ Passing a List as an Argument

def print_fruits(fruit_list):
    for fruit in fruit_list:
        print(fruit)

fruits = ["Apple", "Banana", "Mango"]
print_fruits(fruits)
Analogy: Handing a shopping list to a shopkeeper — they check each item one by one.

✔ Calling a Function

def show_message():
    print("Learning functions is fun!")

show_message()
