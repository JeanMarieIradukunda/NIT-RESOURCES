Python Closures Notes

A. Definition & Explanation

Definition: A closure in Python is a function that remembers values from its enclosing scope, even after the outer function has finished executing. It “closes over” variables from its parent function.

How it works:

  • An outer function defines some variables.
  • An inner function uses these variables.
  • Even if the outer function finishes, the inner function remembers the variables.

Example:

def outer_function(x):
    def inner_function(y):
        return x + y  # inner function remembers x
    return inner_function

add_five = outer_function(5)  # returns inner_function
print(add_five(10))  # Output: 15

Explanation:
outer_function(5) runs and returns inner_function. Even though outer_function is done, inner_function remembers that x = 5. This memory of x is what makes it a closure.

B. Uses and Applications (When to Use)

  • Data hiding / Encapsulation: Variables from the outer function are not accessible globally, protecting them from accidental changes.
  • Maintaining state without global variables: Useful for counters or settings that need to “remember” past values.
  • Avoiding global variables: Keeps your program clean and modular.
  • Functional programming patterns: Often used with decorators or callback functions.

Example (State maintenance):

def counter():
    count = 0
    def increment():
        nonlocal count
        count += 1
        return count
    return increment

visitor_counter = counter()
print(visitor_counter())  # Output: 1
print(visitor_counter())  # Output: 2

C. Real-Life Examples

Creating a Counter for Website Visits:

def visit_counter():
    count = 0
    def increment():
        nonlocal count
        count += 1
        return f"Total visits: {count}"
    return increment

counter = visit_counter()
print(counter())  # Total visits: 1
print(counter())  # Total visits: 2

Implementing User-Specific Settings:

def user_settings(user_name):
    settings = {"theme": "light", "notifications": True}
    def get_settings():
        return f"{user_name}'s settings: {settings}"
    return get_settings

user1 = user_settings("Alice")
print(user1())  # Alice's settings: {'theme': 'light', 'notifications': True}

Tracking Login Attempts in a System:

def login_tracker():
    attempts = 0
    def attempt():
        nonlocal attempts
        attempts += 1
        return f"Login attempts: {attempts}"
    return attempt

tracker = login_tracker()
print(tracker())  # Login attempts: 1
print(tracker())  # Login attempts: 2
✅ Key Takeaways for Beginners:
- Closures remember values from their outer function.
- Useful for state maintenance, data hiding, and avoiding global variables.
- Can be applied to counters, user settings, and tracking attempts in programs.
