Python Generators Notes

A. Definition & Explanation

Definition: A generator in Python is a special type of function that returns an iterator which we can iterate over one value at a time. Instead of returning all values at once like a list, a generator produces values on the fly using the yield keyword.

OR: A Python generator is a special function that returns values one at a time using the yield keyword, instead of returning all values at once.

Simply, A generator is a function that produces values one by one using yield keyword.

How it works:

Normal functions use return to send back a value and exit. Generators use yield to pause the function and save its state, so it can resume later where it left off.

Example:

def simple_generator():
    yield 1
    yield 2
    yield 3

gen = simple_generator()  # create a generator object
print(next(gen))  # Output: 1
print(next(gen))  # Output: 2
print(next(gen))  # Output: 3
# next(gen) now would raise StopIteration

Comparison with Normal Functions and Lists:

Feature Normal Function List Generator
Returns Single value All values at once One value at a time
Memory Usage Low for single value High if list is large Very low, lazy evaluation
State Saving No No Yes, resumes from last yield
Suitable for Small computations Storing data Large datasets, streams

B. Uses and Applications (When to Use)

  • Memory efficiency: Generators produce items one at a time, ideal for very large data.
  • Processing large datasets: Read large files line by line without loading the entire file.
  • Infinite sequences: Generate sequences like Fibonacci numbers without limit.
  • Pipelining / streaming data: Pass data through multiple steps without storing everything in memory.

e. Python Generators – Student-Based Examples

A generator is a special function that produces values one at a time using the yield keyword. Unlike lists, generators do not store all values in memory.

🧑‍🎓 Example 1: Generator with Student Names

def student_names():
    yield "Alice"
    yield "Bob"
    yield "John"

for name in student_names():
    print(name)

Output:

Alice
Bob
John

✔ Values are produced one by one

🧑‍🎓 Example 2: Generator from a Student List

students = ["Alice", "Bob", "John"]

def get_students(students):
    for student in students:
        yield student

for s in get_students(students):
    print(s)

✅ Useful when working with large student lists.

🧑‍🎓 Example 3: Generator for Student Marks

marks = [45, 78, 90, 60]

def pass_marks(marks):
    for m in marks:
        if m >= 50:
            yield m

for p in pass_marks(marks):
    print(p)

Output:

78
90
60

✔ Only passing marks are generated.

🧑‍🎓 Example 4: Generator Expression (Short Form)

Generator expressions work like list comprehensions but use ( ) instead of [ ]. 

marks = [40, 55, 70, 30]

passed = (m for m in marks if m >= 50)

for m in passed:
    print(m)

🧑‍🏫 Real-Life Analogy

Think of a teacher reading marks:

  • Reads one mark at a time
  • Does not memorize all marks at once

That is exactly how a generator works.

✅ Why Use Generators?

  • Memory efficient
  • Faster for large student data
  • Clean and readable code
  • Used in data processing, files, and databases

Example (Memory-efficient for large files):

def read_file_line_by_line(file_path):
    with open(file_path, "r") as file:
        for line in file:
            yield line.strip()  # yields one line at a time

for line in read_file_line_by_line("data.txt"):
    print(line)  # memory-friendly

C. Real-Life Examples (Simple for Beginners)

Generating Numbers:

def even_numbers(limit):
    for n in range(limit):
        if n % 2 == 0:
            yield n

for num in even_numbers(10):
    print(num)  # 0, 2, 4, 6, 8

Reading Large CSV Files Line by Line:

def read_csv(file_path):
    with open(file_path) as f:
        for line in f:
            yield line.split(",")  # yields each row as a list

Simple Fibonacci Generator:

def fibonacci(n):
    a, b = 0, 1
    count = 0
    while count < n:
        yield a
        a, b = b, a + b
        count += 1

for num in fibonacci(5):
    print(num)  # 0, 1, 1, 2, 3

Infinite Natural Numbers:

def natural_numbers():
    n = 1
    while True:
        yield n
        n += 1
