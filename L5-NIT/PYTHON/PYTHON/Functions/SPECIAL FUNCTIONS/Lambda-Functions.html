Python Lambda Functions Notes

a. Definition & Explanation

A lambda function in Python is a small, anonymous function defined without a name using the lambda keyword. Unlike regular functions created with def, lambda functions are limited to a single expression and implicitly return the result of that expression.

Syntax:

lambda arguments: expression
  • lambda: Keyword that creates the function.
  • arguments: Inputs to the function, separated by commas.
  • expression: Single expression executed and returned automatically.
# Normal function
def square(x):
    return x * x

# Lambda function
square_lambda = lambda x: x * x

print(square(5))         # Output: 25
print(square_lambda(5))  # Output: 25

Breakdown of square_lambda = lambda x: x * x:

  • lambda: Keyword for creating an anonymous function.
  • x (Arguments): Input to the lambda function. Multiple arguments possible: lambda x, y: x + y.
  • x * x (Expression): Expression that is evaluated and returned.

Key Differences Between def and lambda:

Feature def Function lambda Function
Syntax def func(args): ... lambda args: expression
Name Has a name Usually anonymous
Body Can have multiple statements Only a single expression
Return Explicit return Implicitly returns expression
Use Cases General-purpose, complex logic Short, quick operations

b. Uses and Applications (When to Use)

Lambda functions are useful for:

  • Anonymous Functions: Avoid writing full def functions for simple tasks.
  • Short Operations: Ideal for one-liners and simple calculations.
  • Functional Programming Tools: Used with map(), filter(), reduce().
# map() with lambda
nums = [1, 2, 3, 4]
squared = list(map(lambda x: x**2, nums))
print(squared)  # Output: [1, 4, 9, 16]

# filter() with lambda
even_nums = list(filter(lambda x: x % 2 == 0, nums))
print(even_nums)  # Output: [2, 4]

# reduce() with lambda
from functools import reduce
sum_all = reduce(lambda a, b: a + b, nums)
print(sum_all)  # Output: 10

c. Real-Life Examples

1. Sorting Student Records by Marks

students = [
    {"name": "Alice", "marks": 85},
    {"name": "Bob", "marks": 92},
    {"name": "Charlie", "marks": 78}
]

# Sort by marks using lambda
sorted_students = sorted(students, key=lambda s: s['marks'], reverse=True)
print(sorted_students)
# Output: [{'name': 'Bob', 'marks': 92}, {'name': 'Alice', 'marks': 85}, {'name': 'Charlie', 'marks': 78}]

2. Calculating Discounts in an Online Shop

# Original prices
prices = [100, 250, 400]

# Apply 10% discount using lambda
discounted_prices = list(map(lambda p: p * 0.9, prices))
print(discounted_prices)  # Output: [90.0, 225.0, 360.0]

3. Filtering Even/Odd Numbers in Financial Data

transactions = [101, 202, 303, 404, 505]

# Filter even transactions
even_transactions = list(filter(lambda x: x % 2 == 0, transactions))
print(even_transactions)  # Output: [202, 404]

# Filter odd transactions
odd_transactions = list(filter(lambda x: x % 2 != 0, transactions))
print(odd_transactions)  # Output: [101, 303, 505]

d. Common Lambda Tools (Student-Based Examples)

Lambda functions are most powerful when combined with Python’s built-in functional tools. These tools help process student data efficiently using lists and dictionaries.

1. map() – Transform Data

Purpose: Applies a function to each item in a list.

marks = [70, 82, 65, 90]

new_marks = list(map(lambda m: m + 5, marks))
print(new_marks)
# Output: [75, 87, 70, 95]

2. filter() – Select Data

Purpose: Filters items based on a condition.

marks = [35, 60, 45, 80, 90]

passed = list(filter(lambda m: m >= 50, marks))
print(passed)
# Output: [60, 80, 90]

3. reduce() – Combine Data

Purpose: Reduces a list to a single value.

from functools import reduce

marks = [70, 80, 65, 85]
total = reduce(lambda a, b: a + b, marks)
print(total)
# Output: 300

4. sorted() with lambda – Custom Sorting

students = [
    {"name": "Alice", "marks": 78},
    {"name": "Bob", "marks": 92},
    {"name": "John", "marks": 65}
]

sorted_students = sorted(students, key=lambda s: s["marks"])
print(sorted_students)

5. max() / min() with lambda

top_student = max(students, key=lambda s: s["marks"])
print(top_student)
# Output: {'name': 'Bob', 'marks': 92}

6. any() and all() with lambda

marks = [60, 75, 40, 80]

failed = any(map(lambda m: m < 50, marks))
print(failed)  # True

all_passed = all(map(lambda m: m >= 50, marks))
print(all_passed)  # False

7. dict() + lambda – Dictionary Transformation

students = {"alice": 78, "bob": 85, "john": 67}

updated = dict(map(lambda s: (s[0].upper(), s[1]), students.items()))
print(updated)

🔑 Key Rule to Remember

Use lambda functions for short and simple logic. If the logic becomes complex or hard to read, always prefer a normal def function.

Summary

Lambda functions are lightweight, one-line functions ideal for simple transformations, filtering, or sorting. They are particularly useful in functional programming patterns where functions are passed as arguments.
