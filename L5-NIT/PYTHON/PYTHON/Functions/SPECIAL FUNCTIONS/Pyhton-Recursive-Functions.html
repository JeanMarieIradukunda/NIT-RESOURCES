Python Recursive Functions Notes

A. Definition & Explanation

A recursive function in Python is a function that calls itself to solve a smaller instance of the same problem. The main idea is to break a complex problem into simpler sub-problems that can be solved using the same logic.

Key Points:

  • A recursive function must have a base case to stop the recursion, otherwise it will continue indefinitely and cause a RecursionError.
  • Each recursive call should move closer to the base case.

Structure of a Recursive Function:

def recursive_function(parameters):
    if base_case_condition:
        return base_case_value
    else:
        # function calls itself with a smaller or simpler input
        return recursive_function(smaller_input)

Example – Factorial of a Number:

def factorial(n):
    if n == 0 or n == 1:  # Base case
        return 1
    else:
        return n * factorial(n - 1)  # Recursive call

print(factorial(5))  # Output: 120

Explanation:
Each call reduces n by 1. The base case n == 0 or n == 1 stops the recursion, preventing infinite calls.

B. Uses and Applications (When to Use)

Recursion is appropriate when:

  • A problem can be divided into similar sub-problems.
  • You can express the solution in terms of itself.
  • Iterative solutions exist, but recursion provides simpler or more elegant code.

Common scenarios include:

  • Traversing hierarchical structures (trees, directories, nested lists)
  • Mathematical sequences (factorials, Fibonacci numbers)
  • Searching and sorting algorithms (binary search, quicksort)
  • Backtracking problems (solving puzzles like Sudoku or mazes)

C. Real-Life Examples

1. Calculating Factorials:

# Already shown above
def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))  # Output: 120

2. Searching Files and Folders:

import os

def list_files(folder):
    for item in os.listdir(folder):
        path = os.path.join(folder, item)
        if os.path.isdir(path):
            list_files(path)  # Recursive call for subfolder
        else:
            print(path)

list_files("/path/to/directory")
✅ Key Takeaways for Beginners:
- Recursive functions call themselves to solve smaller sub-problems.
- Always define a base case to prevent infinite recursion.
- Useful for hierarchical structures, sequences, searching, sorting, and backtracking problems.
