Python Decorators Notes

A. Definition & Explanation

Definition: A decorator in Python is a function that modifies the behavior of another function without changing its code. Decorators allow you to add functionality to a function dynamically.

How it works:

  • A decorator wraps another function.
  • The @decorator_name syntax is a shorthand for passing a function to the decorator.

Basic Example:

def simple_decorator(func):
    def wrapper():
        print("Before the function runs")
        func()
        print("After the function runs")
    return wrapper

@simple_decorator  # Apply decorator to say_hello
def say_hello():
    print("Hello!")

say_hello()

# Output:
# Before the function runs
# Hello!
# After the function runs

Explanation:
say_hello is passed to simple_decorator. wrapper() runs before and after the original function. The original function code is untouched, but its behavior is enhanced.

B. Uses and Applications (When to Use)

Decorators are useful when you want to add functionality to functions without changing them directly:

  • Logging: Keep track of function calls and outputs.
  • Authentication & Authorization: Check user permissions before running a function.
  • Performance Monitoring: Measure how long a function takes to run.
  • Caching / Memoization: Store results of expensive function calls.
  • Validation: Check input values before executing the function.

Example: Instead of repeating logging in every function, you can use one decorator for all functions.

C. Real-Life Examples

Logging Function Execution:

def log_decorator(func):
    def wrapper(*args, **kwargs):
        print(f"Function {func.__name__} is running")
        result = func(*args, **kwargs)
        print(f"Function {func.__name__} finished")
        return result
    return wrapper

@log_decorator
def load_homepage():
    print("Homepage loaded")

load_homepage()

# Output:
# Function load_homepage is running
# Homepage loaded
# Function load_homepage finished

Checking User Authentication:

def require_login(func):
    def wrapper(user_logged_in):
        if user_logged_in:
            return func()
        else:
            return "Access Denied. Please login."
    return wrapper

@require_login
def view_profile():
    return "User Profile Page"

print(view_profile(True))   # User Profile Page
print(view_profile(False))  # Access Denied. Please login.

Measuring Execution Time:

import time

def timer(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"{func.__name__} took {end - start:.4f} seconds to execute")
        return result
    return wrapper

@timer
def compute_sum(n):
    total = 0
    for i in range(n):
        total += i
    return total

compute_sum(1000000)
✅ Key Takeaways for Beginners:
- Decorators wrap functions to enhance or modify behavior.
- They are reusable, clean, and reduce repetitive code.
- Common use cases: logging, authentication, performance measurement, caching.
