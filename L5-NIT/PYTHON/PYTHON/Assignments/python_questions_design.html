Python Practice — Exercises

🧠 Conditional Statements

(3 exercises — use if / elif / else)

Q1Check if a Number is Positive or Negative

Write a Python program that asks the user to enter a number and checks whether it is positive, negative, or zero.

Q2Check for Even or Odd

Write a program that accepts a number from the user and prints whether it is even or odd.

Q3Find the Largest of Two Numbers

Ask the user to input two numbers and print which one is greater or if they are equal.

🔁 Loop Statements

(3 exercises — use for and while)

Q1Print Numbers from 1 to 10

Use a for loop to print numbers from 1 to 10.

Q2Sum of First N Natural Numbers

Write a program that takes a number n from the user and uses a while loop to calculate the sum of the first n numbers.

Q3Print Multiplication Table of a Number

Ask the user to enter a number and print its multiplication table up to 10 using a for loop.

🧭 Jump Statements

(3 exercises — break, continue, pass)

Q1Stop Printing When 5 is Reached

Use a for loop to print numbers from 1 to 10 but stop when the number is 5.

Q2Skip Printing Number 5

Write a for loop that prints numbers from 1 to 10, but skips number 5.

Q3Use pass in a Loop

Write a loop from 1 to 5. If the number is 3, use pass and continue to the next iteration, printing all other numbers.

⚙️ Python Functions

(3 exercises — simple functions with no nesting)

Call the function and pass your own name as an argument.

Q2Square a Number

Write a a program of function named square() that takes a number as a parameter and returns its square.

Q3Find the Average of Two Numbers
