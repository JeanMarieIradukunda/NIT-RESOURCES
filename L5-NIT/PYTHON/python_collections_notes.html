📘 Applying Python Collections

🔹 1. What Are Collections in Python?

Collections are data structures/Assemblies used to store multiple values in a single variable. They help organize, manage, and process data efficiently. Python provides built-in collection types in two main categories: • Basic built-in collections • Collections module specialized types

📂 2. Description of Collection Types

🔹 A. Lists in Python

What is a List?
A list is a collection used to store multiple items in a single variable. Lists are ordered, changeable (mutable), and allow duplicate values.

Creating a List

fruits = ["apple", "banana", "orange"]
print(fruits)

Accessing Elements
Lists use indexing. Index starts at 0.

print(fruits[0])   # apple
print(fruits[1])   # banana
print(fruits[-1])  # orange (last item)

Changing an Item

fruits[1] = "grapes"
print(fruits)

Adding Items

fruits.append("mango")      # Adds to end
fruits.insert(1, "kiwi")    # Adds at specific position
print(fruits)

Removing Items

fruits.remove("banana")   # Removes specific value
fruits.pop()              # Removes last item
fruits.pop(0)             # Removes item at index 0
print(fruits)

Useful List Methods

len(fruits)        # Number of items
fruits.sort()      # Sort alphabetically
fruits.reverse()   # Reverse order
fruits.count("apple")  # Count occurrences
fruits.index("apple")  # Find index position

Looping Through a List

for fruit in fruits:
    print(fruit)

Example: Student Marks List

marks = [60, 75, 80, 90]

total = sum(marks)
average = total / len(marks)

print("Total:", total)
print("Average:", average)

Key Characteristics of Lists

  • Ordered (items have index positions)
  • Mutable (can be changed after creation)
  • Allow duplicate values
  • Can store different data types

🔹 B. Tuples in Python

What is a Tuple?
A tuple is a collection used to store multiple items in a single variable. Tuples are ordered but immutable (cannot be changed after creation).

Creating a Tuple

colors = ("red", "green", "blue")
print(colors)

Accessing Elements
Just like lists, tuples use indexing (starting from 0).

print(colors[0])   # red
print(colors[1])   # green
print(colors[-1])  # blue

Important: Tuples Are Immutable

colors[0] = "yellow"   # ❌ This will cause an error

You cannot modify, add, or remove items once a tuple is created.

Tuple Methods (Limited)
Tuples only have two built-in methods:

colors.count("red")   # Counts occurrences
colors.index("green") # Finds index position

Looping Through a Tuple

for color in colors:
    print(color)

Example: Student Information (Using Tuple)

student = ("Alice", 18, "IT")

print("Name:", student[0])
print("Age:", student[1])
print("Course:", student[2])

Why Use Tuples?

  • Data should not change (fixed values)
  • Faster than lists
  • Used for constant data like coordinates, days of week, etc.

Key Characteristics of Tuples

  • Ordered
  • Immutable (cannot be changed)
  • Allow duplicate values
  • Can store different data types

🔹 C. Dictionaries in Python

What is a Dictionary?
A dictionary is a collection that stores data in key–value pairs. Each key is unique and is used to access its corresponding value.

Creating a Dictionary

student = {
    "name": "John",
    "age": 20,
    "course": "IT"
}

print(student)

Accessing Values

print(student["name"])   # John
print(student["age"])    # 20

Changing Values

student["age"] = 21
print(student)

Adding New Key-Value Pairs

student["grade"] = "A"
print(student)

Removing Items

student.pop("course")   # Removes specific key
del student["age"]      # Deletes key
print(student)

Common Dictionary Methods

student.keys()     # Returns all keys
student.values()   # Returns all values
student.items()    # Returns key-value pairs
student.get("name") # Safe access (avoids error if key not found)
student.update({"age": 22}) # Update dictionary

Looping Through a Dictionary

for key, value in student.items():
    print(key, ":", value)

Example: Students Marks Dictionary

marks = {
    "Alice": 85,
    "Bob": 78,
    "Clara": 92
}

total = sum(marks.values())
average = total / len(marks)

print("Total Marks:", total)
print("Average Marks:", average)

Key Characteristics of Dictionaries

  • Store data as key–value pairs
  • Keys must be unique
  • Mutable (can be changed)
  • Fast lookup using keys
  • Ordered (Python 3.7+ maintains insertion order)

🔹 D. Sets in Python

What is a Set?
A set is a collection that is unordered, unindexed, and does not allow duplicate values. Sets are useful for mathematical operations like union and intersection.

Creating a Set

numbers = {1, 2, 3, 4}
print(numbers)

Duplicate Values Are Automatically Removed

numbers = {1, 2, 2, 3}
print(numbers)   # Output: {1, 2, 3}

Adding and Removing Elements

numbers.add(5)        # Add one item
numbers.remove(2)     # Remove specific item
numbers.discard(10)   # Removes safely (no error if not found)
print(numbers)

Set Operations (Mathematical)

A = {1, 2, 3}
B = {3, 4, 5}

print(A.union(B))         # {1,2,3,4,5}
print(A.intersection(B))  # {3}
print(A.difference(B))    # {1,2}
print(A.symmetric_difference(B)) # {1,2,4,5}

Looping Through a Set

for num in A:
    print(num)

Example: Removing Duplicate Student IDs

student_ids = [101, 102, 103, 101, 102]

unique_ids = set(student_ids)
print(unique_ids)

Key Characteristics of Sets

  • Unordered (no index positions)
  • No duplicate values allowed
  • Mutable (can add/remove items)
  • Fast membership testing
  • Supports mathematical set operations

🔹 E. Frozen Set in Python

What is a Frozen Set?
A frozen set is an immutable version of a set. Once created, you cannot add or remove elements. It is used when you need a fixed collection of unique values.

Creating a Frozen Set

fs = frozenset([1, 2, 3])
print(fs)

Important: Frozen Sets Cannot Be Modified

fs.add(4)      # ❌ Error
fs.remove(1)   # ❌ Error

Why Use Frozen Set?

  • When data must not change
  • Can be used as a dictionary key
  • Useful in mathematical operations

Set Operations (Supported)

A = frozenset([1, 2, 3])
B = frozenset([3, 4, 5])

print(A.union(B))                 # {1,2,3,4,5}
print(A.intersection(B))          # {3}
print(A.difference(B))            # {1,2}
print(A.symmetric_difference(B))   # {1,2,4,5}

Example: Using Frozen Set as Dictionary Key

data = {}

key = frozenset(["Math", "Science"])
data[key] = "Group A"

print(data)

Key Characteristics of Frozen Sets

  • Immutable (cannot be changed)
  • Unordered
  • No duplicate values
  • Supports set operations
  • Can be used as dictionary keys

🔹 F. ChainMap in Python

What is ChainMap?
ChainMap is a class from the collections module. It allows you to combine multiple dictionaries into one view without merging them.

Importing ChainMap

from collections import ChainMap

Creating a ChainMap

dict1 = {"a": 1}
dict2 = {"b": 2}

combined = ChainMap(dict1, dict2)

print(combined)

Accessing Values

print(combined["a"])  # 1
print(combined["b"])  # 2

How It Works
ChainMap searches keys in the first dictionary first. If not found, it checks the next dictionary.

Updating Values

dict1["a"] = 10
print(combined["a"])  # 10

Adding New Items

combined["c"] = 3
print(dict1)  # New items are added to the first dictionary by default

Useful Methods

combined.maps        # Returns list of dictionaries
combined.new_child() # Adds a new dictionary on top
combined.parents     # Returns all but the first dictionary

Example: Student Configuration Settings

defaults = {"language": "English", "level": "Beginner"}
user_settings = {"level": "Advanced"}

settings = ChainMap(user_settings, defaults)

print(settings["language"])  # English
print(settings["level"])     # Advanced

When to Use ChainMap?

  • When combining multiple dictionaries
  • When using default settings with overrides
  • When you want a dynamic view without merging data

Key Characteristics

  • Groups multiple dictionaries together
  • Searches from left to right
  • Does not create a new merged dictionary
  • Part of the collections module

🔹 G. Deque (Double-Ended Queue) in Python

What is a Deque?
A deque (double-ended queue) is a data structure from the collections module. It allows fast insertion and removal of items from both the beginning and the end.

Importing Deque

from collections import deque

Creating a Deque

d = deque([1, 2, 3])
print(d)

Adding Elements

d.append(4)        # Add to the right (end)
d.appendleft(0)    # Add to the left (beginning)

print(d)

Removing Elements

d.pop()       # Remove from the right
d.popleft()   # Remove from the left

print(d)

Other Useful Methods

d.extend([5, 6])         # Add multiple items to the right
d.extendleft([7, 8])     # Add multiple items to the left
d.remove(2)              # Remove specific value
d.reverse()              # Reverse order

Example: Queue System (Students Line)

queue = deque()

queue.append("Alice")
queue.append("Bob")
queue.append("Charlie")

print(queue)

served = queue.popleft()
print("Served:", served)
print("Remaining:", queue)

Why Use Deque?

  • Faster than lists for inserting/removing at both ends
  • Useful for queues and stacks
  • Ideal for real-time data processing

Key Characteristics

  • Supports operations at both ends
  • Thread-safe for basic operations
  • Part of the collections module
  • More efficient than lists for front insertions/removals

📘 ✔ Specialized Tools from the collections Module

The collections module is a built-in Python library that provides specialized container data types beyond lists, tuples, dictionaries, and sets.

🔹 1. Counter

What is Counter?
Counter is a class from the collections module used to count the frequency of items in a list, string, or other iterable.

Importing Counter

from collections import Counter

Example: Counting Items

data = ["a", "b", "a", "c", "b", "a"]
count = Counter(data)

print(count)

Common Methods

count["a"]          # Get count of specific item
count.most_common() # Returns items sorted by frequency
count.elements()    # Returns iterator of elements

Example Use Case: Counting student grades or survey responses.

🔹 2. OrderedDict

What is OrderedDict?
OrderedDict is a dictionary that remembers the order in which keys were inserted. (In modern Python versions, regular dictionaries also maintain insertion order.)

Importing OrderedDict

from collections import OrderedDict

Creating an OrderedDict

od = OrderedDict()
od["a"] = 1
od["b"] = 2

print(od)

Useful Methods

od.move_to_end("a")   # Move key to end
od.popitem()          # Remove last item

Example Use Case: When order of data insertion is important.

🔹 3. defaultdict

What is defaultdict?
defaultdict is a dictionary that provides a default value for missing keys automatically.

Importing defaultdict

from collections import defaultdict

Example with Default Integer

d = defaultdict(int)

d["a"] += 1
print(d)

Other Common Default Types

defaultdict(list)
defaultdict(set)

Example Use Case: Grouping data without checking if key exists.

students = [("IT", "Alice"), ("IT", "Bob"), ("HR", "John")]

groups = defaultdict(list)

for dept, name in students:
    groups[dept].append(name)

print(groups)

📘 ✔ Stack and Queue Operations in Python

Stack and Queue are important data structures used to store and manage data in a specific order.

📌 Stack (LIFO - Last In, First Out)

What is a Stack?
A stack follows the LIFO principle. The last item added is the first one removed.

Common Operations:

  • push → Add item (using append)
  • pop → Remove last item
  • peek → View top item
  • is_empty → Check if stack is empty

Example Using List:

stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

print("Stack:", stack)

# Pop operation
stack.pop()

print("After Pop:", stack)

# Peek (Top Element)
print("Top Element:", stack[-1])

Example Use Case: Undo operations, browser history.

📌 Queue (FIFO - First In, First Out)

What is a Queue?
A queue follows the FIFO principle. The first item added is the first one removed.

Common Operations:

  • enqueue → Add item
  • dequeue → Remove first item
  • front → View first item
  • is_empty → Check if queue is empty

Example Using deque (Recommended):

from collections import deque

queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

print("Queue:", queue)

# Dequeue operation
queue.popleft()

print("After Dequeue:", queue)

# Front Element
print("Front Element:", queue[0])

Example Use Case: Printer queue, student waiting line.

🔎 Difference Between Stack and Queue

  • Stack: LIFO (Last In, First Out)
  • Queue: FIFO (First In, First Out)
  • Stack uses pop()
  • Queue uses popleft() with deque
