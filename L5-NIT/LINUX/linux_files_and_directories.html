🧩 Linux File and Directory Management Notes

Ubuntu Linux OS — Creation, Structure, and Management

1. Creation of Linux Files and Directories

In Ubuntu Linux OS, files and directories can be created using both command-line tools and graphical interfaces. System administrators mostly use the terminal.

🧩 Creating Files

CommandExampleDescription
touchtouch myfile.txtCreates an empty file or updates timestamp.
echoecho "Hello Linux" > hello.txtCreates a file and writes text into it.
catcat > notes.txt (then type and press Ctrl+D)Creates a file with typed content.
nano or vinano report.txtOpens a text editor to create or edit files.

📁 Creating Directories

CommandExampleDescription
mkdirmkdir myfolderCreates a new directory.
mkdir -pmkdir -p /home/user/docs/workCreates nested directories automatically.
🧭 Verify creation with ls -l

2. Description of Linux File System Hierarchy

Ubuntu follows the Filesystem Hierarchy Standard (FHS), a consistent structure organizing directories logically.

DirectoryDescription
/Root directory — top of hierarchy.
/binEssential binaries (ls, cp, mv).
/bootBoot loader files and kernel.
/etcConfiguration files.
/homeUser home directories.
/varVariable data (logs, mail, spool).
/usrUser programs and data.
/tmpTemporary files.

🧩 What is a File System?

A file system is the structure that defines how data is stored, accessed, and managed on a disk.

3. Types of Linux File Systems

1️⃣ Native Linux File Systems

File SystemDescription / Features
ext2Stable, no journaling.
ext3Ext2 + journaling support.
ext4Default in Ubuntu; supports large files, journaling.
XFSHigh performance for enterprise.
BtrfsModern, supports snapshots and RAID.
✅ Most common in Ubuntu: ext4

2️⃣ Non-Native File Systems

File SystemOrigin / Usage
FAT32 / exFATUsed by Windows/USB drives; simple, widely compatible.
NTFSWindows FS, supported via ntfs-3g.
HFS / APFSUsed by macOS systems.

3️⃣ Network File Systems

File SystemDescription
NFSShares files between Linux systems.
SMB/CIFSWindows file-sharing protocol.

4️⃣ Virtual File Systems

File SystemDescription
procfsProcess and system info.
sysfsHardware and kernel info.
tmpfsRAM-based temporary filesystem.

🧱 Summary

CategoryExamplesPurpose
Native Linux FSext4, XFSMain file storage
Non-Native FSFAT32, NTFSCross-platform use
Network FSNFS, SMBRemote access
Virtual FSprocfs, sysfsSystem info

4. Managing Files and Directories

🧭 Viewing and Listing

CommandExampleDescription
lsls -lList files and directories.
pwdpwdShow current working directory.

🔁 Copying, Moving, and Renaming

CommandExampleDescription
cpcp file1.txt /home/user/Copy files.
mvmv old.txt new.txtMove or rename files.

🗑️ Deleting

CommandExampleDescription
rmrm file.txtDelete file.
rm -rrm -r folderDelete directory recursively.

🔒 Permissions

CommandExampleDescription
chmodchmod 755 script.shChange permissions.
chownchown user:group file.txtChange file ownership.

📦 Archiving & Compression

CommandExampleDescription
tartar -cvf archive.tar folder/Create a tar archive.
gzipgzip file.txtCompress file.
