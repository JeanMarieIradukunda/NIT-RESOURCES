A. Respond to the Following Questions (5 marks each)

1. In a school computer lab, the IT technician is testing network connectivity to check whether a student PC can reach the school server and also wants to verify DNS name resolution. Which tools should be used and how do they differ?
 Use ping (connectivity test) and
 nslookup/dig (DNS resolution).
 Ping checks reachability; nslookup/dig checks name-to-IP translation.
2. How to test if a web server is accessible locally?
 Access http://localhost in browser or
 use curl http://localhost.
 Also check service with systemctl status apache2.
3. At a company office, the network administrator needs very detailed DNS records, including authority and additional sections, for troubleshooting a website issue. Which command should be used?
Use dig command for full DNS details including authority and additional sections.
4. In a training center, the IT department wants to confirm that reverse DNS is properly configured for the school mail server’s IP address. What command can be used to verify this?
 Use nslookup -type=PTR or
 dig -x to check PTR record.
5. At PADRI VJEKO TSS, a student cannot open nit.padri-vjeko.tss in the browser, but when typing 192.168.10.20, the website loads successfully. As the school IT officer, what is the most likely problem?
Most likely DNS misconfiguration (wrong or missing A record / DNS server issue).
6. How does encryption improve web security?
Encryption (HTTPS/SSL/TLS) protects data by converting it into unreadable format during transmission.
7. What is a DNS Zone, and what are its different types?
A DNS Zone is a portion of DNS database managed by an authoritative server.
Types:
 Forward
 Reverse
 Primary (Master)
 Secondary (Slave)
 Stub.
8. A user cannot access the website but the server is running. What could be the problem?
 firewall blocking ports,
 wrong configuration,
 service not running,
 incorrect permissions.
9. After installing Apache, the website does not open in the browser. What should you check first?
First check if Apache service is running using systemctl status apache2.
10. A school wants only teachers to upload files to the FTP server, but students should only be allowed to download files. How can the administrator configure access control to achieve this?
 Configure user accounts,
 set file permissions (chmod/chown),
 Disable anonymous upload in vsftpd.conf.
