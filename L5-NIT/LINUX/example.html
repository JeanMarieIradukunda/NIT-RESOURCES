NIT vsftpd Configuration Notes

🏫 Scenario Overview
  • The school NIT has learning materials (PDFs, notes, videos).
  • All materials are stored in NIT-RESOURCES shared folder.
  • TEACHER uploads, deletes, and updates files.
  • 5 STUDENTS connect from their computers to access the resources.
  • FTP is used for easy download/upload of files.
  • Shared folder path: /srv/ftp/shared → NIT-RESOURCES
📘 vsftpd.conf Explanation

🌐 Network & Listening

listen=YES ✅ The FTP server listens on IPv4.
📌 Allows students and teacher to connect using normal IP addresses.

listen_ipv6=NO ✅ Disables IPv6.
📌 Keeps configuration simple for school networks that use IPv4.

🔐 Access Control

anonymous_enable=NO ❌ Prevents unknown users from accessing NIT resources.
📌 Only teacher and students with usernames can login.

local_enable=YES ✅ Allows Linux system users to login.
📌 Teacher and students are Linux users on the server.

✏️ File Management

write_enable=YES ✅ Allows upload, delete, rename.
📌 Teacher can upload new notes.
📌 Students can upload assignments (if folder permissions allow).

local_umask=002 ✅ Sets shared permissions.
📌 Files uploaded by teacher are readable by students.
📌 Student uploads readable by teacher.

🔒 Security (Chroot)

chroot_local_user=YES 🔒 Locks users inside the shared folder.
📌 Students cannot browse system files.
📌 Teacher stays inside NIT-RESOURCES.

allow_writeable_chroot=YES ✅ Prevents login errors.
📌 Allows file upload inside shared folder.

📁 Shared Folder

local_root=/srv/ftp/shared 📂 Sets the NIT-RESOURCES folder.
📌 All users log directly into shared resources directory.
📌 Everyone sees same files.

🔁 Passive Mode (Client Access)

pasv_enable=YES
pasv_min_port=30000
pasv_max_port=31000 🌍 Enables FTP access from different networks.
📌 Students use FileZilla from home or school.
📌 Works behind firewalls and routers.

📊 Logging & Performance

xferlog_enable=YES 📜 Records file transfers.
📌 Teacher can check who downloaded or uploaded files.

log_ftp_protocol=YES 🔍 Logs FTP commands.
📌 Helps admin troubleshoot login or transfer issues.

use_localtime=YES 🕒 Uses local time.
📌 Logs show correct class-time activity.

reverse_lookup_enable=NO ⚡ Speeds up login.
📌 Students connect faster without DNS delay.

🛡️ Security & Limits

hide_ids=YES 🙈 Hides Linux usernames.
📌 Students don’t see system usernames in file listings.

max_clients=50 👥 Limits total FTP connections.
📌 Prevents server overload.

max_per_ip=5 🚦 Limits connections per computer.
📌 Prevents one student from abusing the connection.

✅ Final Result:

  • Easy student access
  • Teacher full control
