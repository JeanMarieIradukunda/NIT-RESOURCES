Learning Outcome 2: Customize Computer Features

Learning Hours: 10

1. Identification of Customer Requirements for an Operating System

Definition: The process of understanding what a user needs from a computer system, including software compatibility, hardware requirements, and intended tasks.

Explanation: This ensures that the computer system chosen meets performance expectations, supports required applications, and is cost-effective for the customer.

Understanding requirements includes checking existing hardware, budget, software needs, and user preferences to avoid incompatibility or wasted resources.

2. Requirements to Consider

When choosing or installing an Operating System, it is important to consider certain requirements to ensure proper performance and compatibility.

a) Budget Constraints

Definition: The amount of money available to buy a computer and operating system.

Example: A student may choose a mid-range laptop with Windows instead of a more expensive computer to reduce costs.

b) Compatibility with Current Hardware

Definition: Making sure the operating system works well with the existing CPU, RAM, storage, and graphics.

Example: A computer with low RAM may use Windows 10 instead of Windows 11.

c) Compatibility with New Hardware

Definition: Ensuring the operating system supports newly installed devices.

Example: Installing a new printer or SSD that works with the OS.

d) How the Computer Will Be Used

Definition: Identifying the main tasks the user will perform on the computer.

Example: A graphic designer needs powerful software, while a student may only need word processing and internet browsing.

e) Compatibility with Existing Applications

Definition: Making sure the software already installed can run on the new operating system.

Example: An office program used in a company must work after upgrading the OS.

f) Types of New Applications to Be Used

Definition: Considering future software that the user plans to install.

Example: If a student plans to use video editing software, the computer must have enough power and support the program.

Identification of Minimum Hardware Requirements for an Operating System

Component Requirement Explanation / Real-Life Example
RAM (Memory) 1 GB (32-bit) / 2 GB (64-bit) Enough memory for basic operations.
Hard Drive 16 GB / 32 GB Minimum disk space for OS files.
Graphics Card DirectX 9 or later Required for display functions.
Processor 1 GHz or faster Handles instructions from OS and apps.
Display 800 x 600 resolution Minimum screen resolution.

Customization of the Operating System

Definition: Operating System (OS) Customization is the process of modifying system settings to improve performance, security, appearance, and user experience.

OS customization allows users to control how the computer looks, works, connects, and performs.

1. System Updates and Security

Definition: System Updates and Security is the process of keeping the operating system updated and protected from threats.

Functions / Purpose:

  • Fix system errors (bugs)
  • Improve performance
  • Protect against viruses and malware
  • Add new features
  • Strengthen firewall protection

Steps to Update Windows:

  • Click Start Menu
  • Open Settings
  • Select Update & Security
  • Click Windows Update
  • Select Check for updates
  • Install available updates
  • Restart computer if required

2. Display Settings

Definition: Display settings control screen appearance such as brightness, resolution, and scaling.

Functions / Purpose:

  • Improve screen visibility
  • Reduce eye strain
  • Adjust screen size
  • Optimize display quality
  • Support different monitor types

Steps to Adjust Display:

  • Open Settings
  • Click System
  • Select Display
  • Adjust brightness, resolution, and scale
  • Click Apply

3. Network and Internet Settings

Definition: Network settings allow the computer to connect to the internet or local networks.

Functions / Purpose:

  • Connect to Wi-Fi
  • Configure Ethernet
  • Set up VPN
  • Manage network security
  • Monitor data usage

Steps to Connect to Wi-Fi:

  • Open Settings
  • Click Network & Internet
  • Select Wi-Fi
  • Turn Wi-Fi ON
  • Choose network
  • Enter password
  • Click Connect

4. Apps and Notifications

Definition: This section manages installed applications and system alerts.

Functions / Purpose:

  • Install or uninstall programs
  • Set default applications
  • Control app permissions
  • Manage notifications
  • Improve system performance

Steps to Manage Apps:

  • Open Settings
  • Click Apps
  • Select installed application
  • Click Uninstall if needed

Steps to Manage Notifications:

  • Open Settings
  • Click System
  • Select Notifications
  • Turn notifications ON or OFF

5. Personalization Settings

Definition: Personalization allows users to change the appearance of the operating system.

Functions / Purpose:

  • Change wallpaper
  • Change themes
  • Enable dark or light mode
  • Customize taskbar
  • Adjust lock screen
  • Improve visual comfort

Steps to Personalize Windows:

  • Open Settings
  • Click Personalization
  • Choose Background, Colors, Themes, or Lock Screen
  • Select desired option
  • Apply changes

Advanced Windows Customization & Management

Definition: Advanced Windows tools are built-in system utilities used to manage storage, automate tasks, monitor system performance, troubleshoot errors, and configure system settings.

Overall Importance:

  • Improve system performance
  • Increase security
  • Organize storage
  • Automate tasks
  • Troubleshoot problems
  • Manage users
  • Customize appearance
  • Maintain system stability

1️⃣ Disk Management

Definition: Disk Management is a built-in Windows tool used to manage storage devices such as hard drives, SSDs, and USB drives.

Purpose:

  • Organize storage space
  • Improve system performance
  • Prepare disks for use
  • Manage partitions
  • Maintain data structure

Use Cases:

  • Installing a new hard drive
  • Separating OS and personal files
  • Preparing a USB drive
  • Fixing storage problems
  • Creating backup partitions

🔹 A) Disk Partitioning

Definition: Dividing a hard drive into separate sections (partitions).

Purpose:

  • Organize data
  • Separate system and personal files
  • Improve file management
  • Enable multi-boot systems

Steps:

  • Right-click Start Menu
  • Click Disk Management
  • Select the disk
  • Right-click unallocated space
  • Click New Simple Volume
  • Follow the wizard
  • Assign drive letter
  • Click Finish

🔹 B) Disk Formatting

Definition: Preparing a storage device by creating a file system and removing existing data.

Purpose:

  • Prepare new drives
  • Remove viruses
  • Change file system (NTFS, FAT32)
  • Fix corrupted drives

Steps:

  • Open Disk Management
  • Right-click the drive
  • Select Format
  • Choose file system
  • Click OK
  • Confirm action

🔹 C) Disk Cleanup

Definition: A tool that removes temporary and unnecessary files.

Purpose:

  • Free storage space
  • Improve speed
  • Remove temporary files
  • Delete recycle bin contents
  • Improve performance

Steps:

  • Search Disk Cleanup
  • Select drive (usually C:)
  • Click OK
  • Select files to delete
  • Click Delete Files

2️⃣ Windows Administrative Tools

Definition: Windows Administrative Tools are advanced system utilities used to monitor, control, automate, and troubleshoot the computer system.

Purpose:

  • System monitoring: Allows users to observe system performance and running processes.
    Example: Checking CPU and memory usage during heavy work.
  • Task automation: Enables automatic execution of system tasks.
    Example: Scheduling daily system backups.
  • Troubleshooting: Helps identify and solve system problems.
    Example: Using logs to find the cause of a program crash.
  • Performance tracking: Monitors system speed and resource usage.
    Example: Detecting slow performance due to high disk usage.
  • Error detection: Records and displays system and application errors.
    Example: Reviewing error messages after software installation failure.

Use Cases:

  • Scheduling antivirus scans
  • Checking system errors
  • Diagnosing system crashes
  • Running automatic backups
  • Monitoring system stability

🔹 A) Task Scheduler

Definition: Task Scheduler is a Windows tool that automatically runs programs, scripts, or tasks at specific times or events.

Purpose:

  • Automate tasks: Allows the computer to perform actions without manual input.
    Example: Automatically running a backup every day.
  • Improve productivity: Saves time by scheduling repetitive tasks.
    Example: Opening a work program automatically at startup.
  • Schedule maintenance: Helps run system maintenance tasks regularly.
    Example: Scheduling antivirus scans every week.

Use Cases:

  • Automatic antivirus scan
  • Daily backup
  • Launching programs at startup
  • Running scripts automatically

Steps:

  • Search Task Scheduler in the Start menu
  • Click Create Basic Task
  • Enter a task name and description
  • Choose a trigger (time, daily, weekly, etc.)
  • Select the required action
  • Choose the program or script to run
  • Click Finish

🔹 B) Event Viewer

Definition: Event Viewer is a Windows tool that records system events, warnings, and errors for monitoring and troubleshooting purposes.

Purpose:

  • Troubleshoot problems: Helps identify the cause of system or application issues.
    Example: Finding why a program is crashing.
  • Monitor system health: Allows users to check the overall performance and stability of the computer.
    Example: Checking if the system has repeated warnings.
  • Analyze errors: Provides detailed error information to help solve technical problems.
    Example: Reviewing error codes after software installation failure.
  • Check security logs: Tracks login attempts and security-related activities.
    Example: Monitoring failed login attempts for security protection.

Steps:

  • Search Event Viewer in the Start menu
  • Click to open the application
  • Expand Windows Logs
  • Select Application, System, or Security
  • Review the listed events, warnings, or errors

3️⃣ Control Panel Settings

Definition: Central system interface used to configure hardware, software, network, and user settings.

Purpose:

  • Configure system settings: Allows users to change important system options such as updates, security, and performance settings.
    Example: Turning on Windows Firewall to improve security.
  • Manage hardware devices: Enables installation and configuration of devices connected to the computer.
    Example: Adding and setting up a printer.
  • Control network connections: Helps users manage internet and local network settings.
    Example: Connecting to Wi-Fi or troubleshooting internet problems.
  • Install/remove programs: Allows users to add new software or uninstall unwanted applications.
    Example: Removing unused software to free disk space.
  • Manage user accounts: Controls user access, passwords, and permissions.
    Example: Creating a new user account for a student.
  • Adjust appearance: Changes how Windows looks, including themes and display options.
    Example: Switching to dark mode or changing desktop wallpaper.

Main Categories:

  • System and Security: Manages updates, firewall, backup, and system protection.
    Example: Checking for Windows updates.
  • Network and Internet: Controls internet connections and network settings.
    Example: Setting up Wi-Fi or configuring a VPN.
  • Hardware and Sound: Manages devices such as printers, speakers, and power settings.
    Example: Adjusting speaker volume or installing a printer.
  • Programs: Allows installation, removal, and management of software.
    Example: Changing the default web browser.
  • User Accounts: Manages users, passwords, and account types.
    Example: Changing a user from Standard to Administrator.
  • Appearance & Personalization: Customizes desktop background, themes, colors, and display settings.
    Example: Changing wallpaper or enabling dark mode.
