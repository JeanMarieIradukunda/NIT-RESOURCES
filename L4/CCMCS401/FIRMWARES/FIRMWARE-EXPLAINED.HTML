Firmware, BIOS, and UEFI Notes

What is Firmware?

Firmware is a type of software that is permanently programmed into a hardware device to control how that device operates. It sits between the hardware and regular software (like your operating system) and provides the low-level instructions that allow the hardware to function properly.

Think of it as the “brain” built into a device that tells it how to work.

Key Points about Firmware:

  1. Embedded Software – Stored on non-volatile memory such as ROM, EEPROM, or flash memory.
  2. Controls Hardware – Directly manages and controls hardware components.
  3. Low-Level Operation – Works closer to the hardware than normal software.
  4. Updates – Sometimes firmware can be updated to fix bugs or add features (e.g., router firmware updates, smartphone updates, BIOS updates).

Examples:

  • BIOS/UEFI on a computer motherboard.
  • Firmware in a printer that controls printing operations.
  • Router firmware that manages network traffic.
  • Firmware in a smartphone that controls touchscreens and sensors.
In short: Firmware = permanent software inside hardware that makes it work.

Firmware vs Operating System

An Operating System (OS) is not firmware, although they are related in how they interact with hardware. Here's the distinction:

Feature Firmware Operating System (OS)
Definition Permanent software programmed into hardware to control device functions. Software that manages hardware resources and provides a platform for applications.
Location Stored in non-volatile memory (ROM, EEPROM, Flash). Stored on storage devices like HDD, SSD, or installed on a computer.
Purpose Enables hardware to function and perform basic tasks. Provides an environment for applications to run and manages hardware resources.
Upgradability Rarely updated, though sometimes firmware updates exist. Regularly updated for features, security, and improvements.
Examples BIOS/UEFI, router firmware, printer firmware. Windows, Linux, macOS, Android.

BIOS and UEFI

BIOS and UEFI are both types of firmware that help a computer start up and manage communication between the operating system and the hardware. They are installed directly on the motherboard.

1. BIOS (Basic Input/Output System)

  • Older technology, used in most PCs before around 2010.
  • Stored in firmware (ROM/flash memory) on the motherboard.
  • Functions:
    • Initializes hardware components (CPU, RAM, storage devices) during boot.
    • Provides a basic interface for hardware settings.
    • Loads the operating system from a bootable device.
  • Limitations:
    • Only supports drives up to 2.2 TB with MBR partitioning.
    • Text-based interface.
    • Slower boot times.

2. UEFI (Unified Extensible Firmware Interface)

  • Modern replacement for BIOS.
  • Functions:
    • All BIOS functions + more advanced features.
    • Supports large drives (>2.2 TB) with GPT partitioning.
    • Graphical interface with mouse support.
    • Faster boot times.
    • Secure Boot to prevent unauthorized software from loading.
  • Advantages over BIOS: More flexible, secure, and capable of handling modern hardware.
In simple terms:
  • BIOS/UEFI = firmware that wakes up your computer and tells it how to start the OS.
  • Think of it like an “on/off switch with instructions” for your PC.
  • UEFI is basically a smarter, faster, safer version of BIOS.
