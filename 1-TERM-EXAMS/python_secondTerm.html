Q3. Analyze Student Marks

Python Code:
marks=[]
for i in range(10):
    while True:
        try:
            m=float(input(f"Mark for student {i+1}: "))
            if 0<=m<=100:
                marks.append(m)
                break
        except ValueError:
            continue

highest = max(marks)
lowest = min(marks)
average = sum(marks)/len(marks)
sorted_marks = sorted(marks)

print("Highest:", highest)
print("Lowest:", lowest)
print("Average:", average)
print("Sorted:", sorted_marks)

Q7. Calculate Bill Function

Python Code:
def calculate_total(items):
    return sum(items)

def calculate_vat(total):
    return total*0.18

def calculate_final(total,vat):
    return total+vat

def calculate_bill(items):
    total = calculate_total(items)
    vat = calculate_vat(total)
    final_amount = calculate_final(total,vat)
    return total, vat, final_amount

items=[100,200,150]
total, vat, final = calculate_bill(items)
print("Total:", total, "VAT:", vat, "Final:", final)
Breaking into smaller functions improves readability, maintainability, and reusability.
