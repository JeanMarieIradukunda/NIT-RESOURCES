SECTION B — ANSWER ALL QUESTIONS (30 Marks)

18. Python data types (10 marks)

  • int: whole numbers, e.g., x = 5
  • float: decimal numbers, e.g., y = 3.14
  • str: text enclosed in quotes, e.g., name = "Alice"
  • bool: True/False, e.g., flag = True
  • list: ordered, mutable collection, e.g., lst = [1,2,3]
  • tuple: ordered, immutable collection, e.g., tup = (1,2)
  • dict: key-value pairs, e.g., d = {"name":"Alice"}
  • set: unordered, unique elements, e.g., s = {1,2,3}
Marking: 1–2 marks per data type with example

19. Python virtual environment steps (10 marks)

  • Create: python -m venv env_name ✅
  • Activate:
    • Windows: env_name\Scripts\activate
    • Linux/Mac: source env_name/bin/activate
  • Use: install packages within environment using pip
  • Deactivate: deactivate
Marking: 2–3 marks per step

21. User-defined functions (10 marks)

  • Syntax: def function_name(parameters):
