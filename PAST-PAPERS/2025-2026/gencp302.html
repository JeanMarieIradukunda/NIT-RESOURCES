21. Decision-Making Architectures 10 Marks
EVALUATION SHEET RESPONSES:

(a) Definition: A switch statement is a multi-way branch selection optimization control syntax that evaluates expression parameters to match integer or character literal constant parameters values to select code branches directly using an internally optimized branch map table lookup layout design mechanism.

(b) Key structural difference: A switch architecture evaluates uniquely scalar targets strictly comparing them with specific fixed identity constant case statements directly, whereas an if-else if framework can evaluate compound, fluid relational equations alongside multifaceted logical expressions containing ranges and varying operand configurations.

(c) Evaluation mapping allocation:

  • Case 1 (Comparing fixed values 1, 2, 3): switch architecture is optimal here because it processes specific literal constants efficiently and provides cleaner layout visibility.

23. Logic Generation: Prime Number Check 15 Marks
EVALUATION SHEET RESPONSES:

(a) Step-by-Step Logic Algorithm:

  1. START program processing.
  2. Read target integer parameter value N.
  3. If N is less than or equal to 1, mark it as NOT PRIME and jump directly to Step 7.
  4. Initialize iterative validation variable counter tracker i = 2.
  5. While i * i <= N, check if N % i == 0. If it is, mark it as NOT PRIME and jump to Step 7. Otherwise, increment i by 1.
  6. If no factor is found through the loop range termination limit, evaluate target as PRIME.
  7. Output result status answer message.
  8. STOP sequence execution.

(b) Full structural program:

#include <stdio.h>
int main() {
    int n, i, isPrime = 1;
    printf("Enter a positive integer: ");
    scanf("%d", &n);
    
    if (n <= 1) {
        isPrime = 0;
    } else {
        for (i = 2; i * i <= n; i++) {
            if (n % i == 0) {
                isPrime = 0;
                break;
            }
        }
    }
    
    if (isPrime == 1)
        printf("%d is a PRIME number.\n", n);
    else
        printf("%d is NOT a PRIME number.\n", n);
        
    return 0;
}
