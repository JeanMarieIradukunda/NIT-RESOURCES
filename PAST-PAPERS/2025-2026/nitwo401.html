20. Network Degradation: Five Primary Impacts of Packet Loss (10 Marks)
1. Latency Overheads: Forces TCP windows to wait for missing segments, ballooning response loops.
2. Retransmission Spikes: Creates heavy retransmission loads, reducing effective network throughput.
3. Jitter & Media Breaks: Causes stuttering and dropped frames in real-time streaming and VoIP traffic.
4. Throughput Collapse: Triggers TCP congestion controls, drastically cutting transmission speeds.
5. Application Session Drops: Causes operational dropouts, data corruption, and connection timeouts.

c & d. Overvoltage Grounding Protection Analysis

Strategic Value: Outdoor equipment on high poles accumulates static charges and attracts lightning strikes. Incorporating inline **Lightning Arrestors** shunts dangerous surge currents to the earth path, saving downstream assets. 22. Mathematical Computation of Signal-to-Noise Ratio (SNR) (10 Marks)
// Mathematical SNR Equation: SNR (dB) = Signal Power (dBm) - Noise Floor Power (dBm)

■ Network Scenario 1 Computation Metrics:

Signal Input = -41 dBm, Noise Floor = -50 dBm

SNR_1 = (-41 dBm) - (-50 dBm) = -41 + 50 = 9 dB

■ Network Scenario 2 Computation Metrics:

Signal Input = -41 dBm, Noise Floor = -96 dBm

SNR_2 = (-41 dBm) - (-96 dBm) = -41 + 96 = 55 dB

Conclusion Evaluation: SNR_2 (55 dB) is significantly greater than SNR_1 (9 dB). Network 2 will deliver clear, fast data transmission, while Network 1 will face severe packet drops due to low signal clarity.

24. Strategic Upgrade Planning & Systematic Troubleshooting Logic (15 Marks)
a & b. Network Upgrade Strategy

Upgrading means replacing old standards (like legacy 802.11a/g) with modern hardware (like Wi-Fi 6/6E or 60GHz wireless bridges). Key considerations before upgrading include available budget, total user capacity, line-of-sight obstacles, and weather-proof hardware ratings.

c. Step-by-Step Troubleshooting Flow

For fixing intermittent outdoor network connections:

  1. Check physical layer cables and PoE injectors for water ingress or loose connections.
  2. Check alignment paths to fix antenna drift from high winds.
  3. Run an RF spectrum scan to spot and resolve channel interference from nearby systems.
  4. Review system logs to catch firmware glitches or IP address conflicts.
