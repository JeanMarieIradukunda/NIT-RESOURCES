SECTION B: Short Answer (Answer ANY 3)

1. Explain the features and advantages of Linux servers.
Open source, stable, secure, supports networking and server applications, flexible configuration.
2. Describe the steps involved in installing a Linux operating system.
Download ISO, create bootable media, boot from media, follow installation prompts, configure partitions, set user and passwords, install additional packages.
3. Explain the Linux file system hierarchy and give examples of important directories.
/ (root), /home, /etc, /var, /usr, /bin, /sbin, /tmp
4. Describe user and group management in Linux including UID and GID.
Users are assigned UIDs, groups have GIDs; users can belong to groups; manage with useradd, groupadd, passwd, and other commands.
5. Explain the three types of file permissions in Linux and how they are set.
Read (r), Write (w), Execute (x); set using chmod or numeric codes (e.g., 755).

SECTION C: Long Answer (Answer ONE)

1. Explain the process of configuring a web server in Linux including installation, configuration files, firewall configuration and security considerations.
Install Apache/Nginx, edit configuration files in /etc, enable firewall ports (80/443), set proper permissions, enable SSL, test server, monitor logs, apply security best practices.
2. Explain how to configure and manage a DNS server in Linux including installation, configuration files, DNS records and testing.
Install BIND, configure named.conf, set zone files, define A, CNAME, MX records, start service, test with dig/nslookup, monitor logs.
