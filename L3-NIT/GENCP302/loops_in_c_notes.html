APPLICATION OF LOOPS IN C PROGRAMMING

🔹 1. Definition of Loops in Programming

A loop is a programming structure that repeats a block of code multiple times until a certain condition is met.

✔️ Examples:
Think of a student practicing multiplication tables: You repeat the same task (like 2 × 1, 2 × 2, 2 × 3...) until you reach 2 × 10. That repetition is exactly what a loop does in programming.

🔹 2. Benefits of Using Loops

Loops help programmers by:

✅ a) Saving Time

Instead of writing the same code many times, we write it once and repeat it.

Example:
If you want to print numbers from 1 to 10:

Without loop:
printf("1\n");
printf("2\n");
...
printf("10\n");

With loop:
for(int i=1; i<=10; i++){
  printf("%d\n", i);
}
✅ b) Reducing Errors

Repeating code manually increases mistakes. Loops make it accurate.

✅ c) Easy to Modify

If you want to change the number of repetitions, you only change one value.

✅ d) Helps in Repetitive Tasks

Examples:

  • Taking attendance every day
  • Practicing past exam papers repeatedly
  • Checking every student’s score

🔹 3. Applications of Loops

Loops are used in many tasks, like:

  • 📌 a) Printing numbers
    • 1 to 100
    • Even numbers
    • Odd numbers
  • 📌 b) Repeating tasks
    • Calculating totals
    • Checking results of many students
    • Summing marks
  • 📌 c) Creating patterns

    Like:

    *
    **
    ***

  • 📌 d) Searching

    Finding a student name in a list.

🔹 4. TYPES OF LOOPS IN C

✅ A) FOR LOOP

Definition
A for loop is a type of loop in C programming that allows you to repeat a block of code a specific number of times. A for loop is used when you know how many times you want to repeat an action. A loop that runs a fixed number of times.

🧠 Examples:
Imagine you have 10 assignments to submit. You decide to complete them one by one from 1 to 10. That’s a for loop.
Syntax
for(initialization; condition; increment/decrement){
  // code to repeat
}
Example
Print numbers 1 to 5:
for(int i=1; i<=5; i++){
  printf("%d\n", i);
}

✅ B) WHILE LOOP

Definition
A while loop is a type of loop in C programming that repeats a block of code as long as a condition is true. A while loop keeps repeating until the condition becomes false.

🧠 Examples:
Imagine you are studying until you understand a topic. You keep studying while you don’t understand.
Syntax
while(condition){
  // code to repeat
}
Example
Print numbers 1 to 5:
int i = 1;
while(i <= 5){
  printf("%d\n", i);
  i++;
}

✅ C) DO-WHILE LOOP

Definition
A do-while loop is a type of loop in C programming that executes the code at least once, and then repeats it as long as the condition is true. Similar to while loop but executes at least once, even if the condition is false. The code inside the loop runs at least once, even if the condition is false.

🧠 Examples:
Imagine you always start reading a book, then decide if you want to continue. You read first, then check.
Syntax
do {
  // code to repeat
} while(condition);
Example
Print numbers 1 to 5:
int i = 1;
do {
  printf("%d\n", i);
  i++;
} while(i <= 5);
✅ Do-While Loop Explained with Bob’s Story
🎯 Scenario
Bob wants to choose the best department to study in.
He is not sure which department is best for him, so he decides to:
✅ Attend at least one class in each department, then decide whether to continue based on what he sees.
🔹 Departments
• Fashion
• ICT
• Carpentry
🔸 Examples (Do-While Loop)
A do-while loop works like this:
1. Bob attends a class once (even if he doesn’t know yet if it is good).
2. After attending, he decides:
o If it is good, he continues.
o If not, he moves to the next department.
So Bob will always attend at least one class, and then decide.
That is exactly how do-while loop works.
✅ C Code Example (Do-While Loop)
#include <stdio.h>
#include <string.h>

int main() {
  char department[20];
  char decision;

  do {
    // Ask Bob which department he wants to attend
    printf("Bob, choose a department (Fashion/ICT/Carpentry): ");
    scanf("%s", department);

    // Show Bob what he learns in the chosen department
    printf("Bob attends %s class once.\n", department);
    printf("After attending, Bob thinks: ");

    // Bob decides whether to continue in the same department
    printf("Do you want to continue in %s? (y/n): ", department);
    scanf(" %c", &decision);

  } while(decision == 'y' || decision == 'Y');

  printf("Bob decided to stop and choose another department.\n");

  return 0;
}
✅ Line-by-Line Explanation
🔹 do { ... } while(condition);
This means:
• Do the actions once first
• Then check the condition
• If true → repeat
• If false → stop
🔹 How it works in the story
1. Bob chooses a department
2. He attends one class
3. He decides if he wants to continue
4. If yes → loop repeats
5. If no → stop

🔹 5. NESTED LOOPS

1️⃣ What Is a Nested Loop?
A nested loop is a loop placed inside another loop.

• The outer loop runs first.
• For each time the outer loop runs, the inner loop runs completely.

🔎 Simple Understanding
If:
• Outer loop runs 3 times
• Inner loop runs 4 times

Then total inner executions = 3 × 4 = 12 times.
Because every time the outer loop runs once, the inner loop runs fully.

2️⃣ Basic Structure in C

for(initialization; condition; update) // Outer loop { for(initialization; condition; update) // Inner loop { // Code to execute } }
You can nest:
• for inside for
• while inside while
• for inside while
• while inside for

🔁 What Is an Iteration?

An iteration is one complete execution of the loop body.
Every time a loop runs once, that is one iteration.

🔹 Example (Single Loop)

#include <stdio.h> int main() { for(int i = 1; i <= 3; i++) { printf("Hello\n"); } return 0; }
Values of i:
• i = 1 → 1st iteration
• i = 2 → 2nd iteration
• i = 3 → 3rd iteration
• i = 4 → condition fails → loop stops

✅ Total iterations = 3

🔁 Understanding Nested Loop Iterations

for(int i = 1; i <= 3; i++) { for(int j = 1; j <= 2; j++) { printf("i=%d j=%d\n", i, j); } }
Outer loop runs 3 times.
Inner loop runs 2 times each time.

Total executions = 3 × 2 = 6 times.

🎯 When Do We Use Nested Loops?

We use nested loops when we repeat something,
and inside that repetition, we repeat something again.

👉 Work inside work.

🧠 Easy Rule:
If you hear:
“For each ___, do something for every ___”
→ You need a nested loop.

⭐ Programming Patterns

What Is a Pattern?
A pattern is something that repeats in a regular and predictable way.
It follows a rule.

🔹 Examples:

1️⃣ Number Pattern
2 4 6 8 10
Rule: Add 2

2️⃣ Shape Pattern
🔵 🔴 🔵 🔴 🔵 🔴
Rule: Alternate colors

3️⃣ Star Pattern
* ** *** ****
Rule: Increase stars step by step.
