FUNCTIONS IN C PROGRAMMING

L3-NIT | Module: GENCP302

Introduction

In C programming, programs can become very long and difficult to understand. To make programs simple, clear, and easy to manage, we use functions. A function helps us divide a big program into small parts.

1. Application of Functions

Definition:
A function in C programming is a block of code that performs a specific task.

Instead of writing the same code again and again, we put it inside a function and call it whenever we need it.

  • To avoid writing the same code many times
  • To make programs easy to read and understand
  • To make programs easy to test and correct errors
  • To divide a big problem into small parts
Real-life Example:
Think of a calculator. Each button (add, subtract, multiply) does one job. Each button is like a function.

2. Types of Functions

There are two main types of functions in C:

1) Library (Built-in) Functions

These are functions already provided by C.

Examples: printf(), scanf()

printf("Hello World");

2) User-Defined Functions

These are functions written by the programmer.

void greet() {
    printf("Hello Student");
}

3. Defining a Function

Defining a function means writing the function code and telling what the function will do.

  • Return type – type of value returned
  • Function name – name of the function
  • Parameters – values passed to the function
  • Function body – statements inside the function
int add(int a, int b) {
    return a + b;
}

4. Function Declaration (Function Prototype)

A function declaration tells the compiler about the function name, return type, and parameters before it is used.

  • C needs to know about a function before calling it
  • Helps avoid errors
int add(int, int);

5. Function Arguments

Arguments are values passed to a function when it is called.

  • Parameters → variables in function definition
  • Arguments → values passed during function call
add(5, 3);

6. Calling a Function

Calling a function means using the function to perform its task. 

int result = add(5, 3);

Complete Example Program

#include <stdio.h>

int add(int, int);   // function declaration

int add(int a, int b) {  // function definition
    return a + b;
}

int main() {
    int sum;
    sum = add(5, 3);   // function call
    printf("Sum = %d", sum);
    return 0;
}

Summary

  • A function is a block of code that does a specific job
  • Functions make programs simple and reusable
  • Main steps:
    1. Declare a function
    2. Define a function
    3. Call a function
