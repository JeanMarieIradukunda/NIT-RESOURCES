✅ C Programming – Conditional Statements

🧠 1. What is a Condition?

A condition is something that evaluates to either:

  • TRUE (1)
  • FALSE (0)

Example:

Click to view code
age >= 18 // TRUE if age is 18 or more, otherwise FALSE

🟢 2. Why use Conditional Statements?

To let the computer do different things depending on the situation.

✔️ Real-life examples:

  • If it’s raining → take an umbrella
  • If you are hungry → eat
  • If your phone battery is low → charge it
  • If traffic light is red → stop, else → go

🟦 3. Types of Conditional Statements in C

There are 5 main types:

  • if
  • if...else
  • else if
  • nested if
  • switch

Let’s explain each with definitions + real-life examples + C code.

🟢 A. if Statement

Definition: Runs a block of code only if a condition is true.

Real-life example: “If the door is open, enter the room.”

Click to view code
int age = 20; if (age >= 18) { printf("You are an adult."); }

🔵 B. if...else Statement

Definition: If TRUE → do something, else → do something different.

Real-life example: “If it rains, take an umbrella; else wear sunglasses.”

Click to view code
int marks = 45; if (marks >= 50) { printf("You passed!"); } else { printf("You failed."); }

🟣 C. else if Ladder

Definition: Used when you have multiple conditions.

Real-life example: “If time is morning → Good morning, Else if afternoon → Good afternoon, Else → Good evening.”

Click to view code
int temperature = 30; if (temperature > 35) { printf("It's very hot."); } else if (temperature > 20) { printf("It's warm."); } else { printf("It's cold."); }

🟠 D. Nested if

Definition: One decision depends on another.

Real-life example: “If you have money, then check if you have time to go shopping.”

Click to view code
int hasMoney = 1; int hasTime = 0; if (hasMoney) { if (hasTime) { printf("You can go shopping."); } else { printf("You have money but no time."); } }

🟡 E. switch Statement

Definition: Check one value against many possible cases.

Real-life example: Phone menu: Press 1 for English, 2 for French, 3 for Swahili

Click to view code
int choice = 2; switch (choice) { case 1: printf("English"); break; case 2: printf("French"); break; case 3: printf("Swahili"); break; default: printf("Invalid choice"); }

🎯 SUMMARY TABLE

Statement Meaning Best Use
if Run code only if condition is TRUE One condition
if...else True → do this, False → do something else Two possibilities
else if Many different conditions Multiple choices
nested if One decision inside another Dependent decisions
switch Check one value with multiple cases Menus, modes, options

🧩 Mini Real Life Problem

Problem: A student scores 78 marks. Show grade:

  • 80–100 → A
  • 70–79 → B
  • 60–69 → C
  • else → Fail
Click to view code
int marks = 78; if (marks >= 80) { printf("Grade A"); } else if (marks >= 70) { printf("Grade B"); } else if (marks >= 60) { printf("Grade C"); } else { printf("Fail"); }
