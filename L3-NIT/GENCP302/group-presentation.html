Group Work Activities

Student Marks Management System

Design a C program that stores student marks, calculates averages, and assigns grades.

Arrays Functions Loops Control Statements
  • Input marks for 10 students
  • Calculate averages
  • Assign grades
  • Display pass/fail students
Employee Salary Processing

Build a salary system that calculates employee earnings, taxes, and net salaries.

Arrays Functions Loops
  • Store employee names
  • Calculate gross salary
  • Deduct tax
  • Display net salary
Library Book Management

Develop a program for storing, searching, and managing library books.

Searching Loops Arrays
  • Store book titles
  • Search books
  • Show availability
  • Display reports
Attendance Monitoring System

Record student attendance and identify students with low attendance.

Arrays Functions Reports
  • Record attendance
  • Count attendance days
  • Generate reports
  • Identify low attendance
Banking Transaction System

Create a banking system for deposits, withdrawals, and balance checking.

Menu Driven Functions Loops
  • Deposit money
  • Withdraw money
  • Check balances
  • Validate transactions
Supermarket Sales Management

Build a sales recording system for products, quantities, and total sales reports.

Arrays Reports Functions
  • Store products
  • Calculate total sales
  • Highest selling product
  • Generate reports

Presentation Checklist

3 Algorithm Design Logical and organized algorithm 10
4 Use of Control Statements Proper use of if, else, switch 10
5 Program Execution Correct outputs and execution 10
6 Documentation Code comments and explanations 10
Total 100 Marks

Presentation Guidelines

Introduce Group Members

Every member should participate and explain their contribution.

Present Flowcharts & Algorithms

Explain the logic and flow of the solution clearly.

Demonstrate Program Execution

Show how the program works with sample outputs.

Answer Questions Confidently

Respond accurately to questions from teachers and classmates.
