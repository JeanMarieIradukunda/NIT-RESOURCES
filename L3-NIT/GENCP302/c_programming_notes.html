🔹 What Does \t Mean in C?

✅ \t = Tab Space

It adds a horizontal space in the output.

It moves the cursor to the next tab position (like pressing the Tab key on a keyboard).

📌 Example:

#include <stdio.h>

int main()
{
    printf("Name\tAge\tGrade\n");
    printf("John\t15\tA\n");
    printf("Mary\t16\tB\n");

    return 0;
}

🖥 Output:

Name Age Grade
John 15 A
Mary 16 B

👉 \t makes the text aligned in columns.

  • Tables
  • Seating plans
  • Reports
  • Grids

🔹 What Does .2f Mean in C?

✅ .2f is used in printf() to format decimal numbers.

It means: Display a floating-point number with 2 digits after the decimal point.

📌 Example:

#include <stdio.h>

int main()
{
    float number = 5.6789;

    printf("Number = %.2f", number);

    return 0;
}

🖥 Output:

Number = 5.68

👉 It rounds the number to 2 decimal places.

🔹 Breaking It Down

  • % → Start of format specifier
  • .2 → Show 2 digits after decimal
  • f → Floating-point number

{

6️⃣ Infinite Loops
