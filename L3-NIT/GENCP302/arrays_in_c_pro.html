📘 ARRAYS IN C PROGRAMMING

Complete Explained Modern Notes

🔹 1. Application of Arrays

✅ What is an Array?

An array is a collection of elements of the same data type stored in continuous memory locations. It allows storing multiple values using a single name.

🧠 Real-Life Analogy

Instead of creating many variables:

int student1;
int student2;
int student3;

You use:

int students[30];

✅ Why Use Arrays?

  • Store many values of same type
  • Process large data efficiently
  • Perform repetitive calculations
  • Organize related data

✅ Applications

int marks[5] = {70,85,60,90,75};
  • Schools
  • Exam systems
  • Report generation

🔹 2. Types of Arrays

1️⃣ One-Dimensional Array

int numbers[5] = {10,20,30,40,50};

Access:

printf("%d", numbers[2]); // Output: 30

2️⃣ Two-Dimensional Array

int matrix[2][3] = { {1,2,3}, {4,5,6} };

Access:

printf("%d", matrix[1][2]); // Output: 6

3️⃣ Multi-Dimensional Arrays

int data[2][3][4];

Used in 3D graphics and simulations.

🔹 3. Declaring Arrays

Syntax:

datatype arrayName[size];

Examples:

int numbers[10];
float prices[5];
char grade[3];

Rules:

  • Size must be positive
  • Index starts from 0
  • Last index = size - 1

🔹 4. Initializing Arrays

Method 1

int numbers[5] = {10,20,30,40,50};

Method 2 (Partial)

int numbers[5] = {10,20};

Method 3

int numbers[] = {1,2,3,4};

Method 4

int numbers[3];
numbers[0]=5;
numbers[1]=10;
numbers[2]=15;

🔹 5. Accessing Elements

int arr[3] = {10,20,30};
printf("%d", arr[1]); // 20

🔹 6. Common Mistakes

  • Using invalid index
  • Forgetting size
  • Mixing data types
  • Not initializing before use

🔹 7. Complete Example Program

#include <stdio.h>

int main() {
  int marks[5];
  int total=0;

  for(int i=0;i<5;i++){
    scanf("%d",&marks[i]);
  }

  for(int i=0;i<5;i++){
    total+=marks[i];
  }

  printf("Total=%d",total);
  printf("Average=%.2f",total/5.0);
  return 0;
}
