Constants in C

In C programming, a constant is a value that cannot be changed during the execution of the program. Once you assign a value to a constant, it stays the same throughout the program. Constants are useful for fixed values like mathematical constants or configuration values.

Types of Constants

  • Integer Constants – Whole numbers (e.g., 10, -5, 0)
  • Floating-point Constants – Decimal numbers (e.g., 3.14, 0.001)
  • Character Constants – Single characters in single quotes (e.g., 'A', '7')
  • String Constants – Sequence of characters in double quotes (e.g., "Hello")
  • Symbolic Constants – Defined using #define or const keyword (e.g., #define PI 3.14 or const int DAYS = 7;)

Assignment Operators

An assignment operator in C is used to store a value into a variable. It takes a value on the right side and assigns it to a variable on the left. The most basic operator is =.

1️⃣ Simple Assignment Operator

int x;
x = 5;   // assigns 5 to x
printf("%d", x); // Output: 5

2️⃣ Compound Assignment Operators

int x = 5;
x += 3;  // x = x + 3
x -= 3;  // x = x - 3
x *= 3;  // x = x * 3
x /= 3;  // x = x / 3
x %= 3;  // x = x % 3

3️⃣ Bitwise Compound Assignment Operators

int x = 5;
x &= 3;  // AND
x |= 3;  // OR
x ^= 3;  // XOR
x >>= 2; // Right shift
x <<= 1; // Left shift
Tip: Before shifting, understand the binary representation. Left shift multiplies by 2 per shift; right shift divides by 2 per shift.

Comparison Operators

Comparison operators compare two values. The result is either 1 (true) or 0 (false).

int x = 5;
int y = 3;
printf("%d", x > y); // Output: 1 (true)

List of Comparison Operators

!=Not equalx != yReturns 1 if values are not equal
>Greater thanx > yReturns 1 if first value is greater
<Less thanx < yReturns 1 if first value is smaller
>=Greater than or equalx >= yReturns 1 if first value is greater or equal
<=Less than or equalx <= yReturns 1 if first value is smaller or equal

Conditional Statements in C

Conditional statements allow the program to make decisions based on a condition.

1️⃣ if statement

int marks = 75;
if (marks >= 50) {
    printf("You passed the exam.\n");
}

2️⃣ if-else statement

int marks = 40;
if (marks >= 50) {
    printf("You passed the exam.\n");
} else {
    printf("You failed the exam.\n");
}

3️⃣ else-if ladder

int marks = 65;
if (marks >= 80) {
    printf("Grade: A\n");
} else if (marks >= 60) {
    printf("Grade: B\n");
} else {
    printf("Grade: C\n");
}

4️⃣ switch statement

int day = 3;
switch(day) {
    case 1: printf("Monday\n"); break;
    case 2: printf("Tuesday\n"); break;
    case 3: printf("Wednesday\n"); break;
    default: printf("Another day\n");
}
