C PROGRAM STRUCTURE – DETAILED NOTES

1. Preprocessor Directives

A preprocessor is a program that processes source code before it's compiled

Preprocessor directives tell the computer what to prepare before the program runs, such as including libraries.

Directives : Commands / Instructions / Directions

Example

#include <stdio.h>

Explanation

#include adds ready-made code from libraries.

<stdio.h> allows input and output like printf() and scanf().

Applications

Printing text on the screen

Reading user input

Using math functions

Usage

Used at the top of the program, before main().

2. Main Function

The main() function is the starting point of every C program.

Example

int main() {
    return 0;
}

Explanation

Program execution starts here.

return 0 means the program ended successfully.

Applications

Controls the entire program

Runs all statements and functions

Usage

Every C program must have exactly one main() function.

3. Statements and Expressions

Statements are instructions that tell the computer what to do.

Example

printf("Hello\n");

Explanation

Tells the computer to print text.

Ends with a semicolon ;.

Applications

Calculations

Printing messages

Assigning values

Usage

Written inside functions, mainly inside main().

4. Variables and Data Types

Variables store data values in memory.

Example

int age = 20;

Explanation

int → type of data (number)

age → variable name

20 → value stored

Applications

Storing user input

Calculating results

Holding data for processing

Usage

Declared before using them in a program.

5. Input and Output Functions

Input/output functions allow the program to communicate with the user.

Example

scanf("%d", &age);
printf("Age: %d", age);

Explanation

scanf() gets input from the user.

printf() shows output on the screen.

Applications

Reading names, numbers, marks

Displaying results and messages

Usage

Requires #include <stdio.h>.

6. Control Structures

(a) Conditional Statements

Control structures help the program make decisions.

Example

if (age >= 18) {
    printf("Adult");
}

Explanation

Checks a condition.

Runs code only if the condition is true.

Applications

Checking pass/fail

Validating passwords

Age verification

Usage

Used when decisions are needed.

(b) Loops

A loop is a programming construct that allows you to repeat a set of instructions multiple times

Loops repeat a block of code multiple times.

Example

for (int i = 0; i < 5; i++) {
    printf("Hello\n");
}

Explanation

Runs the same code again and again.

Saves time and code space.

Applications

Printing lists

Counting numbers

Repeating tasks

Usage

Used when repetition is needed.

7. Functions

Functions are small blocks of code that perform a specific task.

Example

void greet() {
    printf("Hello!");
}

Explanation

Makes the program organized.

Can be reused many times.

Applications

Calculations

Reusable tasks

Large programs

Usage

Called inside main() or other functions.

8. Comments

Comments explain the code for humans. The computer ignores them.

Example

// This prints a message

Explanation

Helps programmers understand the code.

Useful for learning and teamwork.

Applications

Code explanation

Debugging

Documentation

Usage

Used anywhere in the program.

Complete Example Using All Structures

#include <stdio.h>

// Main function
int main() {

    int age;

    printf("Enter your age: ");
    scanf("%d", &age);

    if (age >= 18) {
        printf("You are an adult.\n");
    }

    return 0;
}
