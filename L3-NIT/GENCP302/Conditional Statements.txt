
		CONDITIONAL STATEMENTS

A conditional statement in C is a way for a program to make decisions. 
It allows the program to execute certain parts of code only if a specific condition is true, and optionally execute other code if the condition is false.

	3 basic types of conditional statements
	---------------------------------------
 1. if statement ---> Executes a block of code if the condition is true.
The condition in an if statement is a logical test — an expression that checks whether something is true or false.

Syntax
if (condition) {
  // block of code to be executed if the condition is true
}

2. else statement --> Use the else statement to specify a block of code to be executed if the condition is false.

Syntax
if (condition) {
  // block of code to be executed if the condition is true
} else {
  // block of code to be executed if the condition is false
}
