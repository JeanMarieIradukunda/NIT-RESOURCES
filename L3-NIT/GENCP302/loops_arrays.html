Programming Problems: Loops & Arrays

A. Application of Loops

1. For Loop

Problem: Print all even numbers between 1 and 50.

Algorithm:
Start → Set i = 1 → Repeat while i ≤ 50 → If i % 2 == 0, display i → Increment i → Stop

2. While Loop

Problem: Calculate sum of numbers until user enters 0.

Algorithm:
Start → Initialize sum = 0 → Input number → While number ≠ 0 → Add number to sum → Input again → Display sum → Stop

3. Do-While Loop

Problem: Check if number is positive or negative until -1 is entered.

Algorithm:
Start → Repeat: Input number → If number > 0 display "Positive" → If number < 0 and not -1 display "Negative" → Until number = -1 → Stop

4. Nested Loops (Pattern)

Problem: Display star pattern.

Algorithm:
Start → Outer loop (1 to 5) → Inner loop (1 to row number) → Print * → New line → Stop

5. Nested Loops (Multiplication Table)

Problem: Display tables from 1 to 10.

Algorithm:
Start → Outer loop (1 to 10) → Inner loop (1 to 10) → Multiply values → Display result → Stop

B. Application of Arrays

6. One-Dimensional Array

Problem: Store and display 5 integers.

Algorithm:
Start → Declare array of size 5 → Input values using loop → Display values → Stop

7. Reverse Array

Problem: Store 10 integers and display in reverse.

Algorithm:
Start → Declare array size 10 → Input values → Display from last index to first → Stop

8. Initialize Array & Average

Problem: Calculate average of {10,20,30,40,50}.

Algorithm:
Start → Initialize array → Set sum = 0 → Add all elements → Average = sum / total elements → Display result → Stop

9. Find Largest Element

Problem: Find largest number in array.

Algorithm:
Start → Input array values → Assume first value as largest → Compare with others → Update if larger found → Display largest → Stop

10. Two-Dimensional Array (3×3 Matrix)

Problem: Declare and display matrix.

Algorithm:
Start → Declare 3×3 array → Use nested loops to input values → Use nested loops to display matrix → Stop
