APPLICATION OF CONDITIONAL STATEMENTS

Conditional statements help a program make decisions, just like humans do in daily life.

1. IF STATEMENT

Definition

The if statement checks a condition and performs an action only if the condition is true.

Usage

Used when only one condition needs to be checked.

When to Use

  • When there is one decision
  • When no alternative action is required

Syntax

if (condition) {
    statement;
}

Simple Example (C Program)

int money = 500;

if (money >= 300) {
    printf("You can buy food");
}

Real-Life Example

👉 If you have enough money, you buy food.

Applications

  • Checking pass marks
  • Checking age eligibility
  • Verifying login access

2. ELSE IF STATEMENT

Definition

The else if statement checks multiple conditions one after another.

Usage

Used when more than one condition needs to be tested.

When to Use

  • When there are many choices
  • When decisions depend on different conditions

Syntax

if (condition1) {
    statement;
}
else if (condition2) {
    statement;
}
else {
    statement;
}

Simple Example (C Program)

int marks = 65;

if (marks >= 80) {
    printf("Grade A");
}
else if (marks >= 60) {
    printf("Grade B");
}
else {
    printf("Fail");
}

Real-Life Example

👉 If score is high → Grade A

👉 If score is average → Grade B

👉 Otherwise → Fail

Applications

  • Grading systems
  • Salary categories
  • Weather decisions

3. SWITCH STATEMENT

Definition

The switch statement selects one option from many choices based on a value.

Usage

Used when you compare one variable with many fixed values.

When to Use

  • When there are menu options
  • When values are fixed (1, 2, 3…)

Syntax

switch (variable) {
    case value1:
        statement;
        break;
    case value2:
        statement;
        break;
    default:
        statement;
}

Simple Example (C Program)

int day = 1;

switch (day) {
    case 1:
        printf("Monday");
        break;
    case 2:
        printf("Tuesday");
        break;
    default:
        printf("Invalid day");
}

Real-Life Example

👉 Press 1 → Call

👉 Press 2 → Message

👉 Any other → Invalid option

Applications

  • ATM menus
  • Phone menu systems
  • School timetable selection

4. NESTED CONDITIONS

Definition

Nested conditions are if statements inside another if statement.

Usage

Used when a decision depends on another decision.

When to Use

  • When conditions are dependent
  • When checking step by step

Syntax

if (condition1) {
    if (condition2) {
        statement;
    }
}

Simple Example (C Program)

int age = 20;
int hasID = 1;

if (age >= 18) {
    if (hasID == 1) {
        printf("You can vote");
    }
}

Real-Life Example

👉 If you are 18 years

👉 AND you have an ID

👉 THEN you can vote

Applications

  • Job eligibility checks
  • Security systems
  • Loan approval
