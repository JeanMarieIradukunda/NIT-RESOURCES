C Programming Assignment Questions

1. Multiple Choice Questions (MCQs)

  1. Which of the following is not a token in C?
    • a) Keyword
    • b) Operator
    • c) Variable
    • d) Preprocessor directive
  2. Which data type in C is used to store decimal values?
    • a) int
    • b) float
    • c) char
    • d) double
  3. The symbol # in C is used for:
    • a) Variables
    • b) Preprocessor directives
    • c) Comments
    • d) Loops
  4. Which of the following is a valid constant declaration in C?
    • a) const int x = 5;
    • b) int const x = 5;
    • c) #define PI 3.14
    • d) All of the above
  5. What is the purpose of the return statement in a function?
    • a) To print output
    • b) To end the program
    • c) To return a value to the caller
    • d) To declare a variable
  6. Which operator is used for logical AND in C?
    • a) &
    • b) &&
    • c) |
    • d) ||
  7. Which of the following if statements will execute when x = 10?
    • a) if(x < 5)
    • b) if(x == 10)
    • c) if(x > 20)
    • d) if(x != 10)
  8. To call a function named sum with two arguments a and b, the correct syntax is:
    • a) sum(a, b);
    • b) call sum(a, b);
    • c) function sum(a, b);
    • d) sum = (a, b);
  9. What is the output of the following code? 
    int x = 5;
if(x > 0)
    printf("Positive");
else
    printf("Negative");
    • a) Positive
    • b) Negative
    • c) 5
    • d) Error
  10. Which of the following is incorrect about functions in C?
    • a) A function can have zero or more arguments
    • b) A function must always return a value
    • c) Functions help modularize code
    • d) Functions can call other functions

5. Programming-Related Questions

  1. Complete the code to declare an integer variable num and initialize it with 10: 
    ________ num = 10;
  2. Complete the code for a main function that prints "Hello": 
    #include <stdio.h>
int main() {
    ___________("Hello");
    return 0;
}
  3. Predict the output: 
    int a = 5, b = 2;
printf("%d", a / b);
  4. Predict the output: 
    int x = 7;
if(x % 2 == 0)
    printf("Even");
else
    printf("Odd");
  5. Detect the error: 
    int main() {
    int a = 5
    printf("%d", a);
    return 0;
}
  6. Detect the error: 
    void main() {
    printf("Hello");
}
  7. Write a function add that takes two integers and returns their sum.
  8. Write a switch statement that prints "Red", "Green", or "Blue" depending on color = 2.
  9. Write a program that checks if a number is positive, negative, or zero.

6. Higher-Order / Real-World Programming

  1. Analyze and fix the logical error: 
    int main() {
    int x = 10;
    if(x = 5)
        printf("x is 5");
    return 0;
}
  2. Evaluate which function type is appropriate for a task that prints a message but returns nothing.
  3. Create a program that calculates factorial using a function.
  4. Modify a program to include constants for maximum and minimum allowed values.
  5. Optimize a for loop that prints numbers from 1 to 100.
