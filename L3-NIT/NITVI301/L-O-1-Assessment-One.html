Learning Outcome One – Assessment One

Introduction to VoIP Systems

1. Multiple Choice Questions

1. Which VoIP element is responsible for compressing and decompressing voice data?

  • A. IP-PBX
  • B. Router
  • C. SIP Server
  • D. Codec
Answer: D. Codec
Explanation: Codecs compress voice into digital packets for transmission and decompress them back into audio at the receiver’s end.

2. What is the main benefit of using VoIP in businesses compared to traditional telephone networks?

  • A. It increases the cost of calls
  • B. It requires no Internet connection
  • C. It reduces communication costs and allows advanced features
  • D. It works only on mobile phones
Answer: C. It reduces communication costs and allows advanced features
Explanation: VoIP lowers communication expenses, supports features like call forwarding, voicemail-to-email, and conferencing.
2. Open-Ended Questions

3. Define the term VoIP and give any three types of it.

Answer:
Definition: VoIP (Voice over Internet Protocol) is a technology that allows voice communication and multimedia sessions to be transmitted over IP-based networks instead of traditional telephone networks.
Three Types:
1. IP Telephony (On-Premise VoIP) – Example: Asterisk
2. Hosted VoIP (Cloud-Based VoIP) – Example: RingCentral
3. Mobile VoIP – Example: WhatsApp calls

4. List at least four benefits of using VoIP.

Answer:
1. Reduced communication costs
2. Easy scalability and flexibility
3. Advanced features (call forwarding, voicemail-to-email, conferencing)
4. Mobility – can be used anywhere with Internet
Other possible answers: Efficient use of network infrastructure, integration with data applications

5. Identify three essential elements of a VoIP system and briefly explain their role.

Answer:
1. IP Phones: Hardware or software phones used to capture and transmit voice.
2. IP-PBX: Central system that manages calls and routes them within the network.
3. VoIP Gateway: Connects the VoIP network to the traditional PSTN network.
Other possible elements: SIP Server (handles call signaling), Router & Switches (network connectivity), Codec (compresses/decompresses voice)
