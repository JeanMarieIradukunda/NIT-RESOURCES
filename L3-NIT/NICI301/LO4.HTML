LEARNING OUTCOME 4: OPERATE CCTV SYSTEM

4.1: MONITOR LIVE FOOTAGE OF CAPTURE EVENTS
────────────────────────────────────────

Live footage is the real-time video captured by CCTV cameras and displayed immediately on a monitor, phone, or computer.
It allows users to watch activities as they happen without waiting for recorded playback.

IMPORTANCE OF LIVE FOOTAGE
• Helps monitor security in real time  
• Allows quick response to emergencies  
• Helps detect suspicious activities immediately  
• Improves safety in homes, schools, offices, and industries  

Example
-------
A security guard can watch people entering a building through live camera video displayed on the monitor.

TYPES OF MONITORS FOR CCTV
CCTV systems use different types of monitors to display camera footage.


1. LED MONITOR
------------------------------------------------
An LED monitor is a flat screen display commonly used in modern CCTV systems.

Features:
• Clear picture quality  
• Low power consumption  
• Thin and lightweight  
• Long lifespan  

Use:
Mostly used in offices, schools, shops, and homes.

2. LCD MONITOR
------------------------------------------------
LCD monitors are used to display live video from CCTV cameras.

Features:
• Good image quality  
• Affordable cost  
• Suitable for indoor surveillance systems  

3. HDMI MONITOR OR TV
------------------------------------------------
Modern DVRs and NVRs support HDMI connection to monitors or televisions.

Features:
• High-definition display  
• Better image clarity  
• Supports multiple camera views  

Example:
A 32-inch HD television can display live footage from several cameras at the same time.

4. VGA MONITOR
------------------------------------------------
A VGA monitor uses VGA cable connection for video display.

Features:
• Common in older CCTV systems  
• Lower image quality than HDMI  

5. MOBILE PHONE AND TABLET DISPLAYS
------------------------------------------------
Modern CCTV systems allow live viewing using smartphones and tablets through mobile applications such as Hik-Connect.

Features:
• Remote monitoring from anywhere  
• Easy access through internet  
• Real-time alerts and notifications  

6. COMPUTER MONITOR
------------------------------------------------
A desktop or laptop computer can be used for live CCTV monitoring.

Features:
• Supports multiple camera channels  
• Easy playback and recording management  
• Useful in control rooms  

VIEWING LIVE FOOTAGE
────────────────────────────────────────
Viewing live footage means displaying and monitoring real-time camera video through a DVR, NVR, monitor, computer, or mobile device.

STEPS FOR VIEWING LIVE FOOTAGE ON DVR/NVR

1. POWER ON THE SYSTEM
Turn on:
• Cameras  
• DVR/NVR  
• Monitor  

2. CONNECT CAMERAS
Ensure cameras are properly connected using:
• Network cable (IP cameras)  
• Coaxial cable (analog cameras)  

3. OPEN LIVE VIEW
After login, select the Live View option on the DVR/NVR screen.

4. SELECT CAMERA CHANNEL
Choose the camera channel to display.
Example:
• Channel 1 → Main gate  
• Channel 2 → Office  
• Channel 3 → Parking area  

5. ADJUST DISPLAY SETTINGS
Users can:
• Zoom images  
• Change screen layout  
• Switch between single and multiple camera views  
• Adjust brightness and contrast  

6. REMOTE VIEWING
Live footage can also be viewed remotely using:
• Mobile phones  
• Tablets  
• Computers connected to the internet  

MULTI-SCREEN LIVE VIEWING
Modern CCTV systems support multiple viewing modes.

Single Screen View:
Displays one camera at a time.

Split Screen View:
Displays several cameras on one monitor.

Examples:
• 4-screen view  
• 8-screen view  
• 16-screen view  

IMPORTANCE:
• Monitors many locations at once  
• Improves surveillance efficiency  

ADVANTAGES OF LIVE CCTV MONITORING
• Real-time security monitoring  
• Fast emergency response  
• Better crime prevention  
• Easy remote access  
• Improved safety management  

4.2 MAINTAIN CCTV SYSTEM CONTROL ROOM
────────────────────────────────────────
DESCRIPTION OF CONTROL ROOM
A control room is a secure area where all CCTV monitoring, recording, and management activities are carried out.
It contains monitors, DVR/NVR systems, networking devices, and operators who supervise security.

A control room is a centralized location where CCTV live footage is monitored, recorded data is managed, and security operations are controlled.

FUNCTIONS OF THE CONTROL ROOM
• Monitoring live CCTV footage  
• Managing recordings and playback  
• Responding to security incidents  
• Controlling camera settings and system access  
• Storing and protecting surveillance data  
• Coordinating security responses  

CONTROL ROOM OPERATOR
A control room operator is a trained person responsible for:
• Watching live camera feeds  
• Detecting suspicious activity  
• Operating DVR/NVR systems  
• Exporting video evidence when needed  
• Reporting incidents to security teams or management  

ENVIRONMENTAL CONDITIONS OF CONTROL ROOM

1. Control Room Arrangement
• Monitors should be placed at eye level  
• Equipment should be organized and labeled clearly  
• DVR/NVR systems should be protected in a secure rack  
• Clean and clutter-free workspace is required  

2. Air Conditioning System
• Keeps equipment cool and prevents overheating  
• Maintains stable temperature for electronic devices  
• Improves operator comfort and concentration  

3. Electricity Supply
• Stable power supply is required  
• Use of UPS (Uninterruptible Power Supply) is important  
• Protects system from power failures and voltage fluctuations  

4. Equipment Status
• Regular checking of cameras and recording devices  
• Ensure hard disk is not full or faulty  
• Confirm all cameras are online and functioning  
• Monitor network connection status  

4.3 PERFORM EXPORT / PLAYBACK OF REQUIRED INFORMATION
────────────────────────────────────────
DEFINITION

Export:
Export is the process of copying recorded CCTV video or images from the system to an external storage device such as a USB drive or hard disk.

Playback:
Playback is the process of viewing recorded video footage from the CCTV system at a later time.

STEPS FOR PLAYBACK
1. Log into DVR/NVR system  
2. Select Playback Menu  
3. Choose camera channel  
4. Select date and time  
5. Click Play  
6. Use controls:
   • Pause  
   • Fast forward  
   • Rewind  
   • Zoom  

STEPS FOR EXPORTING VIDEO EVIDENCE
1. Insert USB flash drive or external storage  
2. Open Export Menu on DVR/NVR  
3. Select camera channel  
4. Choose date and time range  
5. Preview footage  
6. Click Export  
7. Wait until transfer is complete  
8. Safely remove storage device  

4.4 INTERPRETATION OF CCTV SYSTEM LOGS
────────────────────────────────────────

DEFINITION OF LOGS
System logs are automatic records created by CCTV systems that show system activities, events, and errors.
They help track what happens inside the CCTV system.

DESCRIPTION OF SYSTEM LOG METRICS

1. User Activity Logs
• Login and logout times  
• User access details  
• Changes made in system settings  

2. Recording Logs
• Start and stop recording times  
• Recording mode changes  
• Storage usage information  

3. Alarm/Event Logs
• Motion detection alerts  
• Intrusion detection  
• Sensor activation events  

4. System Error Logs
• Camera offline alerts  
• Hard disk failure warnings  
• Network connection problems  

5. Device Status Logs
• Camera status (online/offline)  
• DVR/NVR performance status  
• Storage health status  

IMPORTANCE OF LOGS
• Helps detect security issues  
• Assists in troubleshooting system faults  
• Provides evidence of system usage  
• Improves system maintenance  

4.5 GENERATE AND INTERPRET INCIDENT REPORT
────────────────────────────────────────

DEFINITION OF INCIDENT REPORT
An incident report is a written document that records details of a security event captured by CCTV cameras.
It provides clear information about what happened, when it happened, and who was involved.

HOW TO GENERATE AN INCIDENT REPORT
1. Identify the incident from live view or playback  
2. Note the exact date and time  
3. Review and confirm video evidence  
4. Export relevant footage if needed  
5. Record key details in a report format  
6. Attach screenshots or video clips  
7. Submit report to management or security team  

WHAT AN INCIDENT REPORT INCLUDES
• Date and time of incident  
• Location of event  
• Camera reference/channel  
• Description of what happened  
• People or objects involved  
• Duration of the event  
• Actions taken  
• Evidence attached (video/images)  

INTERPRETING AN INCIDENT REPORT
Interpreting means analyzing the report to understand the event clearly.

Key Points:
• What caused the incident?  
• Who was involved?  
• Was it accidental or intentional?  
• What security measures failed or worked?  
• What actions should be taken?  

IMPORTANCE OF INCIDENT REPORTS
• Helps in investigations  
• Provides legal evidence  
• Improves security planning  
• Helps prevent future incidents  
• Supports decision-making in security management
